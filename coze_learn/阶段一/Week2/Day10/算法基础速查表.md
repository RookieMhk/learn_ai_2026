## 一、基础数据结构

### 1. 数组 (Array)
| 维度 | 内容 |
|------|------|
| **原理** | 连续内存空间存储相同类型元素，支持随机访问 |
| **Python实现** | `list`（动态数组）、`array.array`、`numpy.ndarray` |
| **时间复杂度** | 访问O(1)，插入/删除O(n) |
| **空间复杂度** | O(n) |
| **应用场景** | 数值计算、图像处理、缓存实现 |
| **面试要点** | 二维数组遍历、前缀和、差分数组 |

```python

# 中间插入/删除
```

### 2. 链表 (Linked List)
| 维度 | 内容 |
|------|------|
| **原理** | 节点通过指针连接，不需要连续内存空间 |
| **Python实现** | 自定义`Node`类或使用`collections.deque` |
| **时间复杂度** | 访问O(n)，插入/删除O(1)（已知位置时） |
| **空间复杂度** | O(n) |
| **应用场景** | LRU缓存、多项式表示、内存管理 |
| **面试要点** | 反转链表、检测环、合并有序链表 |

```python
class ListNode:
    def __init__(self, val=0, next=None):
        self.val = val
        self.next = next

# 创建链表 1→2→3
head = ListNode(1)
head.next = ListNode(2)
head.next.next = ListNode(3)

# 遍历链表 O(n)
def traverse(head):
    curr = head
    while curr:
        print(curr.val)
        curr = curr.next

# 反转链表 O(n)
def reverse_list(head):
    prev = None
    curr = head
    while curr:
        next_node = curr.next
        curr.next = prev
        prev = curr
        curr = next_node
    return prev
```

```python
stack.append(2)
stack.append(3)

# 出栈

# 栈的应用：括号匹配
def is_valid_parentheses(s):
    stack = []
    mapping = {')': '(', ']': '[', '}': '{'}
    for char in s:
        if char in mapping:
            top = stack.pop() if stack else '#'
            if mapping[char] != top:
                return False
        else:
            stack.append(char)
    return not stack
```

```python
    queue = deque([start])
    visited.add(start)
```

### 5. 哈希表 (Hash Table)
| 维度 | 内容 |
|------|------|
| **原理** | 通过哈希函数将键映射到存储位置，解决冲突（链地址法/开放寻址） |
| **Python实现** | `dict`、`set` |
| **时间复杂度** | 平均O(1)，最坏O(n)（哈希冲突严重时） |
| **空间复杂度** | O(n) |
| **应用场景** | 缓存、词频统计、去重 |
| **面试要点** | 哈希函数设计、冲突解决、LRU缓存实现 |

```python
# 哈希表基本操作
hash_map = {}

# 插入 O(1)平均
hash_map['apple'] = 1
hash_map['banana'] = 2

# 查找 O(1)平均
value = hash_map.get('apple')  # 1

# 删除 O(1)平均
del hash_map['banana']

# 自定义哈希表实现（简化版）
class MyHashMap:
    def __init__(self, size=1000):
        self.size = size
        self.buckets = [[] for _ in range(size)]
    
    def _hash(self, key):
        return hash(key) % self.size
    
    def put(self, key, value):
        bucket_idx = self._hash(key)
        for i, (k, v) in enumerate(self.buckets[bucket_idx]):
            if k == key:
                self.buckets[bucket_idx][i] = (key, value)
                return
        self.buckets[bucket_idx].append((key, value))
    
    def get(self, key):
        bucket_idx = self._hash(key)
        for k, v in self.buckets[bucket_idx]:
            if k == key:
                return v
        return -1
```

### 6. 堆 (Heap)
| 维度 | 内容 |
|------|------|
| **原理** | 完全二叉树，父节点值 ≥（≤）子节点值（大顶堆/小顶堆） |
| **Python实现** | `heapq`模块、`queue.PriorityQueue` |
| **时间复杂度** | 建堆O(n)，插入O(log n)，删除堆顶O(log n) |
| **空间复杂度** | O(n) |
| **应用场景** | 优先队列、Top K问题、堆排序 |
| **面试要点** | 数据流中位数、合并K个有序链表 |

```python
import heapq

# 小顶堆操作
min_heap = []

# 入堆 O(log n)
heapq.heappush(min_heap, 3)
heapq.heappush(min_heap, 1)
heapq.heappush(min_heap, 2)

# 查看堆顶 O(1)
print(min_heap[0])  # 1

# 出堆 O(log n)
smallest = heapq.heappop(min_heap)  # 1

# 应用：Top K问题
def top_k_frequent(nums, k):
    from collections import Counter
    count = Counter(nums)
    
    # 使用最小堆维护前K个
    heap = []
    for num, freq in count.items():
        heapq.heappush(heap, (freq, num))
        if len(heap) > k:
            heapq.heappop(heap)
    
    return [num for freq, num in heap]
```

### 7. 二叉树 (Binary Tree)
| 维度 | 内容 |
|------|------|
| **原理** | 每个节点最多两个子节点（左/右） |
| **Python实现** | 自定义`TreeNode`类 |
| **时间复杂度** | 遍历O(n)，查找O(log n)（平衡时） |
| **空间复杂度** | O(n) |
| **应用场景** | 表达式树、决策树、文件系统 |
| **面试要点** | 前/中/后序遍历、层序遍历、二叉树直径 |

```python
class TreeNode:
    def __init__(self, val=0, left=None, right=None):
        self.val = val
        self.left = left
        self.right = right

# 二叉树遍历
def preorder(root):
    """前序遍历：根→左→右"""
    if not root:
        return []
    return [root.val] + preorder(root.left) + preorder(root.right)

def inorder(root):
    """中序遍历：左→根→右"""
    if not root:
        return []
    return inorder(root.left) + [root.val] + inorder(root.right)

def postorder(root):
    """后序遍历：左→右→根"""
    if not root:
        return []
    return postorder(root.left) + postorder(root.right) + [root.val]

# 层序遍历（BFS）
def level_order(root):
    if not root:
        return []
    
    result = []
    queue = deque([root])
    
    while queue:
        level_size = len(queue)
        level = []
        for _ in range(level_size):
            node = queue.popleft()
            level.append(node.val)
            if node.left:
                queue.append(node.left)
            if node.right:
                queue.append(node.right)
        result.append(level)
    
    return result
```

## 三、排序算法

### 8. 快速排序 (Quick Sort)
| 维度 | 内容 |
|------|------|
| **原理** | 分治法，选择基准分区，递归排序子序列 |
| **Python实现** | 递归或迭代实现 |
| **时间复杂度** | 平均O(n log n)，最坏O(n²)（已排序时） |
| **空间复杂度** | O(log n)递归栈 |
| **应用场景** | 大规模数据排序、K-th元素查找 |
| **面试要点** | 分区函数、优化（三数取中）、尾递归优化 |

```python
def quick_sort(arr):
    """快速排序 O(n log n)平均"""
    if len(arr) <= 1:
        return arr
    
    pivot = arr[len(arr) // 2]  # 选择基准
    left = [x for x in arr if x < pivot]
    middle = [x for x in arr if x == pivot]
    right = [x for x in arr if x > pivot]
    
    return quick_sort(left) + middle + quick_sort(right)

# 原地分区版本（节省空间）
def partition(arr, low, high):
    """分区函数，返回基准位置"""
    pivot = arr[high]  # 选择最右元素为基准
    i = low - 1
    
    for j in range(low, high):
        if arr[j] <= pivot:
            i += 1
            arr[i], arr[j] = arr[j], arr[i]
    
    arr[i + 1], arr[high] = arr[high], arr[i + 1]
    return i + 1

def quick_sort_inplace(arr, low=0, high=None):
    """原地快速排序 O(log n)空间"""
    if high is None:
        high = len(arr) - 1
    
    if low < high:
        pi = partition(arr, low, high)
        quick_sort_inplace(arr, low, pi - 1)
        quick_sort_inplace(arr, pi + 1, high)
```

### 9. 归并排序 (Merge Sort)
| 维度 | 内容 |
|------|------|
| **原理** | 分治法，递归分割到最小单元，再合并有序序列 |
| **Python实现** | 递归实现，需要额外空间 |
| **时间复杂度** | O(n log n)稳定 |
| **空间复杂度** | O(n) |
| **应用场景** | 链表排序、外部排序、逆序对计数 |
| **面试要点** | 合并函数、链表归并排序、逆序对问题 |

```python
def merge_sort(arr):
    """归并排序 O(n log n)"""
    if len(arr) <= 1:
        return arr
    
    mid = len(arr) // 2
    left = merge_sort(arr[:mid])
    right = merge_sort(arr[mid:])
    
    return merge(left, right)

def merge(left, right):
    """合并两个有序数组 O(n)"""
    result = []
    i = j = 0
    
    while i < len(left) and j < len(right):
        if left[i] <= right[j]:
            result.append(left[i])
            i += 1
        else:
            result.append(right[j])
            j += 1
    
    result.extend(left[i:])
    result.extend(right[j:])
    return result

# 链表归并排序（常考）
def merge_sort_linked_list(head):
    if not head or not head.next:
        return head
    
    # 快慢指针找中点
    slow, fast = head, head.next
    while fast and fast.next:
        slow = slow.next
        fast = fast.next.next
    
    mid = slow.next
    slow.next = None  # 断开链表
    
    left = merge_sort_linked_list(head)
    right = merge_sort_linked_list(mid)
    
    return merge_linked_lists(left, right)
```

### 10. 堆排序 (Heap Sort)
| 维度 | 内容 |
|------|------|
| **原理** | 利用堆性质，反复取出堆顶元素 |
| **Python实现** | 基于`heapq`或自定义堆 |
| **时间复杂度** | O(n log n) |
| **空间复杂度** | O(1)原地排序 |
| **应用场景** | 需要原地排序的场景、实时数据排序 |
| **面试要点** | 堆化过程、原地排序实现 |

```python
def heap_sort(arr):
    """堆排序 O(n log n)原地"""
    n = len(arr)
    
    # 建堆 O(n)
    for i in range(n // 2 - 1, -1, -1):
        heapify(arr, n, i)
    
    # 逐个取出堆顶 O(n log n)
    for i in range(n - 1, 0, -1):
        arr[0], arr[i] = arr[i], arr[0]  # 交换堆顶和末尾
        heapify(arr, i, 0)  # 对剩余元素堆化
    
    return arr

def heapify(arr, n, i):
    """维护堆性质 O(log n)"""
    largest = i  # 假设当前节点最大
    left = 2 * i + 1
    right = 2 * i + 2
    
    # 检查左子节点
    if left < n and arr[left] > arr[largest]:
        largest = left
    
    # 检查右子节点
    if right < n and arr[right] > arr[largest]:
        largest = right
    
    # 如果需要交换
    if largest != i:
        arr[i], arr[largest] = arr[largest], arr[i]
        heapify(arr, n, largest)  # 递归调整
```

## 四、搜索算法

### 11. 二分查找 (Binary Search)
| 维度 | 内容 |
|------|------|
| **原理** | 在有序数组中，每次比较中间元素，缩小搜索范围一半 |
| **Python实现** | 迭代或递归实现 |
| **时间复杂度** | O(log n) |
| **空间复杂度** | O(1)迭代，O(log n)递归 |
| **应用场景** | 有序数据查找、数值计算求根、范围查询 |
| **面试要点** | 边界条件、查找插入位置、旋转数组查找 |

```python
def binary_search(arr, target):
    """二分查找 O(log n)"""
    left, right = 0, len(arr) - 1
    
    while left <= right:
        mid = left + (right - left) // 2  # 防止溢出
        
        if arr[mid] == target:
            return mid
        elif arr[mid] < target:
            left = mid + 1
        else:
            right = mid - 1
    
    return -1  # 未找到

# 查找第一个等于target的位置
def binary_search_first(arr, target):
    left, right = 0, len(arr) - 1
    result = -1
    
    while left <= right:
        mid = left + (right - left) // 2
        if arr[mid] == target:
            result = mid
            right = mid - 1  # 继续在左侧查找
        elif arr[mid] < target:
            left = mid + 1
        else:
            right = mid - 1
    
    return result
```

### 12. 深度优先搜索 (DFS)
| 维度 | 内容 |
|------|------|
| **原理** | 沿着树的深度遍历，尽可能深地搜索分支 |
| **Python实现** | 递归或栈实现 |
| **时间复杂度** | O(V+E)（顶点+边） |
| **空间复杂度** | O(h)递归深度 |
| **应用场景** | 路径查找、连通分量、拓扑排序 |
| **面试要点** | 回溯法、排列组合、数独求解 |

```python
# 递归DFS
def dfs_recursive(graph, node, visited=None):
    if visited is None:
        visited = set()
    
    visited.add(node)
    print(node)
    
    for neighbor in graph[node]:
        if neighbor not in visited:
            dfs_recursive(graph, neighbor, visited)

# 栈实现DFS
def dfs_iterative(graph, start):
    visited = set()
    stack = [start]
    
    while stack:
        node = stack.pop()
        if node not in visited:
            visited.add(node)
            print(node)
            
            # 将未访问的邻居入栈
            for neighbor in reversed(graph[node]):  # 保持顺序
                if neighbor not in visited:
                    stack.append(neighbor)

# 应用：全排列
def permutations(nums):
    def backtrack(path, used):
        if len(path) == len(nums):
            result.append(path[:])
            return
        
        for i in range(len(nums)):
            if not used[i]:
                used[i] = True
                path.append(nums[i])
                backtrack(path, used)
                path.pop()
                used[i] = False
    
    result = []
    backtrack([], [False] * len(nums))
    return result
```

### 13. 广度优先搜索 (BFS)
| 维度 | 内容 |
|------|------|
| **原理** | 逐层遍历，先访问所有相邻节点，再深入下一层 |
| **Python实现** | 队列实现 |
| **时间复杂度** | O(V+E) |
| **空间复杂度** | O(w)队列宽度 |
| **应用场景** | 最短路径、层级遍历、社交网络 |
| **面试要点** | 最短路径问题、层序遍历、双向BFS |

```python
from collections import deque

def bfs(graph, start):
    visited = set([start])
    queue = deque([start])
    
    while queue:
        node = queue.popleft()
        print(node)
        
        for neighbor in graph[node]:
            if neighbor not in visited:
                visited.add(neighbor)
                queue.append(neighbor)

# 应用：最短路径（无权图）
def shortest_path_bfs(graph, start, end):
    if start == end:
        return [start]
    
    visited = {start}
    queue = deque([(start, [start])])  # (节点, 路径)
    
    while queue:
        node, path = queue.popleft()
        
        for neighbor in graph[node]:
            if neighbor == end:
                return path + [neighbor]
            
            if neighbor not in visited:
                visited.add(neighbor)
                queue.append((neighbor, path + [neighbor]))
    
    return []  # 无路径
```
