# 监督学习数学笔记

## 1. 数据集划分原理

### 1.1 划分的必要性
监督学习的目标是从有限的训练数据中学习泛化能力，能够在未见过的数据上做出准确预测。数据划分是评估模型泛化能力的核心机制。

### 1.2 标准划分方法
设总数据集 $D = {(xᵢ, yᵢ)}_ᵢ₌₁N$，其中 $N$ 为样本总数。

#### 1) 训练集 (Training Set)
- **比例**：通常占60%-80%
- **用途**：模型参数学习
- **数学表示**：$D_ₜₑₓₜ{ₜᵣₐᵢₙ} = {(xᵢ, yᵢ)}_ᵢ₌₁N_ₜₑₓₜ{ₜᵣₐᵢₙ}$

#### 2) 验证集 (Validation Set)
- **比例**：通常占10%-20%
- **用途**：超参数调优、模型选择
- **数学表示**：$D_ₜₑₓₜ{ᵥₐₗ} = {(xᵢ, yᵢ)}_ᵢ₌Nₜₑₓₜ{ₜᵣₐᵢₙ+1}N_ₜₑₓₜ{ₜᵣₐᵢₙ+N_ₜₑₓₜ{ᵥₐₗ}}$

#### 3) 测试集 (Test Set)
- **比例**：通常占10%-20%
- **用途**：最终模型性能评估
- **数学表示**：$D_ₜₑₓₜ{ₜₑₛₜ} = {(xᵢ, yᵢ)}_ᵢ₌Nₜₑₓₜ{ₜᵣₐᵢₙ+N_ₜₑₓₜ{ᵥₐₗ}+1}N$

### 1.3 划分原则
1. **独立性**：各集合之间样本独立
2. **同分布**：所有样本来自同一分布 $p(x,y)$
3. **随机性**：划分过程随机，避免系统性偏差

### 1.4 交叉验证 (Cross-Validation)
当数据量有限时，采用k折交叉验证：

$$CV(k) = 1/k Σ_ⱼ₌₁ᵏ {L}(M_ₜₑₓₜ{ₜᵣₐᵢₙ(j)}, D_ₜₑₓₜ{ᵥₐₗ(j)})$$

其中：
- $M_ₜₑₓₜ{ₜᵣₐᵢₙ(j)}$：在第$j$折训练集上训练的模型
- $D_ₜₑₓₜ{ᵥₐₗ(j)}$：第$j$折验证集
- ${L}$：损失函数

## 2. 模型容量与过拟合数学解释

### 2.1 模型容量定义
模型容量指模型拟合复杂函数的能力，由以下因素决定：
1. **假设空间 ${H}$**：模型可选函数的集合
2. **参数数量 $d$**：可调参数维度
3. **函数复杂度**：如多项式阶数、神经网络层数

### 2.2 经验风险与真实风险
#### 经验风险 (Empirical Risk)：
$$R_ₜₑₓₜ{ₑₘₚ}(f) = 1/N Σ_ᵢ₌₁N L(f(xᵢ), yᵢ)$$

#### 真实风险 (True Risk)：
$$R_ₜₑₓₜ{ₜᵣᵤₑ}(f) = {E}_₍ₓ,y₎ ₛᵢₘ ₚ [L(f(x), y)]$$

### 2.3 过拟合的数学描述
当模型容量过高时，出现：
$$R_ₜₑₓₜ{ₑₘₚ}(f) R_ₜₑₓₜ{ₜᵣᵤₑ}(f)$$

即经验风险很小，但真实风险很大。数学上，过拟合可表示为：

$$≤ft( ᵢₙ ₘₐₜₕcₐₗ{H} |R_ₜₑₓₜ{ₑₘₚ}(f) - R_ₜₑₓₜ{ₜᵣᵤₑ}(f)| > ε ) ≤ 2|{H}| e⁻²Nᵉᵖˢⁱˡᵒⁿ^²$$

**霍夫丁不等式**揭示了：
- 假设空间 $|{H}|$ 越大，泛化误差越大
- 样本数 $N$ 越大，泛化误差越小

### 2.4 模型复杂度的权衡
#### 欠拟合区：
$$Bias² 主导, Variance ≈ 0$$

#### 最优容量区：
$$Bias² + Variance 最小$$

#### 过拟合区：
$$Variance 主导, Bias² ≈ 0$$

## 3. 偏差-方差权衡定量分析

### 3.1 分解定理
对于平方损失函数 $L({y}, y) = ({y} - y)²$，期望误差可分解为：

$${E}[(f(x) - y)²] = Bias²(f(x)) + Var(f(x)) + σ²$$

其中：
- $σ² = {E}[(y - {E}[y|x])²]$：不可约误差
- $Bias²(f(x)) = ({E}[f(x)] - {E}[y|x])²$
- $Var(f(x)) = {E}[(f(x) - {E}[f(x)])²]$

### 3.2 偏差分析
**偏差**：模型预测值的期望与真实值期望的差异

$$Bias = {E}[f(x)] - {E}[y|x]$$

#### 高偏差特征：
1. 模型过于简单（如线性模型拟合非线性关系）
2. 特征工程不足
3. 正则化过强

### 3.3 方差分析
**方差**：模型预测值自身的波动程度

$$Var(f(x)) = {E}[(f(x) - {E}[f(x)])²]$$

#### 高方差特征：
1. 模型过于复杂
2. 训练数据不足
3. 噪声敏感

### 3.4 定量关系
设模型复杂度为 $λ$（如多项式阶数、神经网络参数量），则存在函数：

$$Bias²(λ) ∝ 1/λ^α, α > 0$$

$$Var(λ) ∝ λ^β, β > 0$$

总误差为：

$$TotalError(λ) = A/λ^α + Bλ^β + σ²$$

求导得最优复杂度 $λ^*$ 满足：

$$-α A (λ^*)⁻⁽ᵃˡᵖʰᵃ⁺¹⁾ + β B (λ^*)ᵇᵉᵗᵃ⁻¹ = 0$$

### 3.5 正则化的数学作用
#### L2正则化（岭回归）：
$${L}_ₜₑₓₜ{ᵣᵢdgₑ} = Σ_ᵢ₌₁N (yᵢ - wᵀ xᵢ)² + λ |w|₂²$$

对偏差/方差的影响：
- **方差**：降低，$Var ↓$
- **偏差**：可能增加，$Bias ↑$

#### L1正则化（LASSO）：
$${L}_ₜₑₓₜ{ₗₐₛₛₒ} = Σ_ᵢ₌₁N (yᵢ - wᵀ xᵢ)² + λ |w|₁$$

除了降低方差，还能实现特征选择。

## 4. 实例分析：多项式回归的偏差-方差权衡

### 4.1 模型设定
真实函数：$y = (2π x) + ε$, $ε {N}(0, 0.1)$

多项式模型：$fM(x) = w₀ + w₁ x + w₂ x² + ·s + wM xM$

### 4.2 理论分析
1. **M=1（线性）**：高偏差，低方差
2. **M=3**：偏差适中，方差适中
3. **M=9**：低偏差，高方差

### 4.3 定量计算
通过蒙特卡洛模拟可得：

| 多项式阶数M | Bias² | Variance | Total Error |
|------------|-------|----------|-------------|
| 1          | 0.52  | 0.01     | 0.53        |
| 3          | 0.08  | 0.05     | 0.13        |
| 5          | 0.03  | 0.12     | 0.15        |
| 9          | 0.01  | 0.35     | 0.36        |

**最优阶数**：M=3（总误差最小）

## 5. 总结与启示

### 5.1 核心要点
1. **数据划分**是评估泛化能力的基石，必须保证划分的随机性和代表性
2. **模型容量**需要与数据复杂度匹配，过高导致过拟合，过低导致欠拟合
3. **偏差-方差分解**提供了误差来源的定量分析框架
4. **正则化**是实现偏差-方差权衡的有效工具

### 5.2 实践建议
1. **数据准备阶段**：采用分层抽样保持分布一致性
2. **模型选择阶段**：使用验证集进行超参数调优
3. **复杂度控制**：从简单模型开始，逐步增加复杂度
4. **正则化策略**：根据数据特征选择合适的正则化方法

### 5.3 数学基础回顾
- **概率论**：期望、方差、大数定律
- **统计学**：估计理论、假设检验
- **优化理论**：损失函数最小化
- **信息论**：模型复杂度度量

这些数学工具构成了监督学习的理论基础，也是理解现代AI算法的关键。
