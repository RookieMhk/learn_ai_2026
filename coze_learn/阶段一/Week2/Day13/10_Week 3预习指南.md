# Week 3 预习指南：深度学习基础入门

## 一、深度学习概述

### 1.1 什么是深度学习
深度学习是机器学习的一个分支，通过多层神经网络学习数据的层次化表示。与传统机器学习相比，深度学习能够自动提取特征，在处理图像、语音、自然语言等复杂数据时表现出色。

### 1.2 深度学习 vs 传统机器学习
| 维度 | 传统机器学习 | 深度学习 |
|------|--------------|----------|
| 特征工程 | 人工设计特征 | 自动学习特征 |
| 数据需求 | 相对较少 | 大量标注数据 |
| 计算资源 | 中等 | 高（GPU加速） |
| 可解释性 | 较好 | 较差（黑盒问题） |
| 应用领域 | 结构化数据 | 非结构化数据 |

### 1.3 深度学习发展里程碑
- 2012年：AlexNet在ImageNet竞赛中夺冠，开启深度学习热潮
- 2014年：生成对抗网络（GAN）提出
- 2017年：Transformer架构革命性进展
- 2020年：大语言模型（LLM）爆发
- 2024年：多模态模型快速发展

## 二、神经网络基础概念

### 2.1 神经元与激活函数
**神经元**：深度学习的基本单元，模拟生物神经元行为

**常见激活函数**：
- **Sigmoid**：σ(x) = 1/(1+e^{-x})，输出(0,1)，易饱和
- **Tanh**：tanh(x) = (e^x - e^{-x})/(e^x + e^{-x})，输出(-1,1)
- **ReLU**：f(x) = max(0,x)，计算简单，缓解梯度消失
- **Leaky ReLU**：f(x) = max(αx, x)，解决"神经元死亡"问题
- **Softmax**：将输出转换为概率分布，用于多分类

### 2.2 前向传播
**定义**：输入数据通过网络层计算得到输出的过程

**数学表示**：
```
z[l] = W[l] · a[l-1] + b[l]
a[l] = g[l](z[l])
```
其中：
- W[l]：第l层的权重矩阵
- b[l]：第l层的偏置向量
- g[l]：第l层的激活函数
- a[l]：第l层的激活值

### 2.3 反向传播与梯度下降
**反向传播**：通过链式法则计算损失函数对每个参数的梯度

**核心步骤**：
1. 前向传播计算输出
2. 计算损失函数
3. 反向计算各层梯度
4. 使用梯度下降更新参数

**梯度下降更新公式**：
```
W = W - η · ∂L/∂W
b = b - η · ∂L/∂b
```

## 三、深度学习框架简介

### 3.1 PyTorch vs TensorFlow
| 特性 | PyTorch | TensorFlow |
|------|---------|------------|
| 开发方 | Meta（原Facebook） | Google |
| 动态图 | 默认动态图（Eager Execution） | 2.x版本支持动态图 |
| 社区生态 | 研究领域主导 | 工业部署成熟 |
| 学习曲线 | 相对平缓 | 较陡峭 |
| 移动端部署 | Torch Mobile | TensorFlow Lite |

### 3.2 框架核心概念
**张量（Tensor）**：深度学习中的基本数据结构，多维数组

**计算图（Computational Graph）**：描述计算过程的有向无环图

**自动微分（Autograd）**：自动计算梯度的机制

**优化器（Optimizer）**：实现各种优化算法的组件

### 3.3 推荐学习路径
1. **Week 3-4**：PyTorch基础（张量操作、自动微分、简单网络）
2. **Week 5-6**：TensorFlow基础与对比学习
3. **Week 7+**：根据兴趣方向选择主攻框架

## 四、计算机视觉基础

### 4.1 卷积神经网络（CNN）
**核心思想**：通过卷积核提取局部特征，通过池化降低维度

**典型结构**：
- 卷积层：提取特征
- 池化层：下采样，降低计算量
- 全连接层：最终分类

**经典网络**：
- LeNet-5（1998）：手写数字识别
- AlexNet（2012）：ImageNet竞赛冠军
- VGG（2014）：简单规则的网络结构
- ResNet（2015）：残差连接，解决深度网络训练难题

### 4.2 图像处理基础
**数据增强**：旋转、裁剪、翻转、颜色变换等，增加数据多样性

**迁移学习**：使用预训练模型，在特定任务上微调

## 五、自然语言处理基础

### 5.1 循环神经网络（RNN）
**特点**：处理序列数据，具有记忆能力

**问题**：长序列梯度消失/爆炸

**改进**：LSTM、GRU门控机制

### 5.2 Transformer架构
**核心创新**：自注意力机制，并行计算

**BERT**：双向编码表示，掩码语言模型

**GPT**：单向自回归模型，生成任务

## 六、算法与数据结构基础

### 6.1 核心数据结构
- **数组**：连续存储，随机访问O(1)
- **链表**：非连续存储，插入删除O(1)
- **栈**：LIFO后进先出
- **队列**：FIFO先进先出
- **树**：二叉树、二叉搜索树、堆
- **图**：邻接矩阵、邻接表表示

### 6.2 核心算法
- **排序**：快速排序、归并排序、堆排序
- **搜索**：二分查找、DFS、BFS
- **动态规划**：背包问题、最长公共子序列
- **贪心算法**：霍夫曼编码、最小生成树
