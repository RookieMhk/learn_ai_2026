```
优化算法 (Optimization Algorithms)
│
├── 🔹 一阶方法 (First-order)

│

│       └── L-BFGS

└── 🔹 自适应方法 (Adaptive)
    ├── AdaGrad
    ├── RMSProp
    ├── Adam
    ├── AdamW
    └── NADAM
```

## 📊 三、模型评估指标

### 3.1 分类指标体系
```
分类评估指标
│
├── 🎯 基础指标
│   ├── 准确率 (Accuracy)
│   ├── 精确率 (Precision)
│   ├── 召回率 (Recall)
│   └── F1分数 (F1-Score)
│
├── 📈 综合指标
│   ├── ROC-AUC
│   ├── PR-AUC (AP)
│   └── 混淆矩阵 (Confusion Matrix)
│
└── 📉 损失函数
    ├── 交叉熵 (Cross-Entropy)
    ├── 铰链损失 (Hinge Loss)
    └── Focal Loss
```

### 3.2 回归指标体系
```
回归评估指标
│
├── 📐 距离度量
│   ├── MSE (均方误差)
│   ├── RMSE (均方根误差)
│   └── MAE (平均绝对误差)
│
├── 📊 统计度量
│   ├── R² (决定系数)
│   ├── Adjusted R²
│   └── MAPE (平均绝对百分比误差)
│
└── 📉 鲁棒度量
    └── Huber Loss
```

### 3.3 评估指标选择指南
```
┌─────────────────────────────────────────────────────────────┐
│                     指标选择决策树                          │
├─────────────────────────────────────────────────────────────┤
│  Q1: 类别平衡?                                              │
│      ├─ Yes → Accuracy                                     │
│      └─ No  → 继续Q2                                        │
│                                                             │
│  Q2: 更关心正例识别?                                        │
│      ├─ Yes → Precision-Recall分析                         │
│      └─ No  → 继续Q3                                        │
│                                                             │
│  Q3: 需要概率输出?                                          │
│      ├─ Yes → AUC-ROC / LogLoss                            │
│      └─ No  → F1-Score / Cohen's Kappa                      │
└─────────────────────────────────────────────────────────────┘
```

特征工程完整流程

├── 1️⃣ 数据预处理

## ⏱️ 五、算法复杂度分析

### 5.1 时间复杂度速查表
```
┌─────────────────────────────────────────────────────────────┐
│              常用算法时间复杂度速查表                        │
├────────────────┬────────────┬──────────────────────────────┤
│     算法       │   训练复杂度  │          预测复杂度        │
├────────────────┼────────────┼──────────────────────────────┤
│ KNN            │ O(1)       │ O(n·d)                       │
│ 线性回归       │ O(n·d²)   │ O(d)                         │
│ Logistic回归   │ O(n·d·k)  │ O(d)                         │
│ 决策树         │ O(n·d·log n)│ O(log n)                    │
│ 随机森林       │ O(m·n·d·log n)│ O(m·log n)              │
│ SVM (核方法)   │ O(n²·d)~O(n³)│ O(d·s) (s=支持向量数)    │
│朴素贝叶斯      │ O(n·d)     │ O(d·|C|)                     │
└────────────────┴────────────┴──────────────────────────────┘
注: n=样本数, d=特征数, m=树数量, k=迭代次数, |C|=类别数
```

### 5.2 空间复杂度分析
```
空间复杂度考量
│
├── 📦 模型参数存储
│   ├── 线性模型: O(d)
│   ├── 树模型: O(d·m) (m=树的深度)
│   └── 神经网络: O(#weights)
│
├── 📊 训练过程存储
│   ├── 梯度: O(#parameters)
│   ├── 一阶矩估计: O(#parameters)
│   └── 二阶矩估计: O(#parameters)
│
└── 📈 数据存储
    ├── 原始数据: O(n·d)
    └── 增强数据: O(k·n·d)
```

### 5.3 Big-O 复杂度图
```
        │
   指数 │              O(2^n)
   增长  │
   速率  │          O(n!)
            │
            │      O(n^3)
            │
            │  O(n^2)
            │
            │O(n log n)
            │
            │ O(n)
            │
            │ O(log n)
            │
            │ O(1)
            └─────────────────────────────→
                     输入规模(n)
```
