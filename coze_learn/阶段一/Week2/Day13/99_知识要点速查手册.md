# 知识要点速查手册

## 一、监督学习核心概念

### 1.1 线性回归
- **定义**：通过线性组合特征预测连续值的模型
- **数学原理**：ŷ = w₁x₁ + w₂x₂ + ... + wₙxₙ + b，目标是最小化损失函数 L(w, b) = (1/n)∑(y_i - ŷ_i)²
- **Python实现要点**：
  ```python
  from sklearn.linear_model import LinearRegression
  model = LinearRegression()
  model.fit(X_train, y_train)
  predictions = model.predict(X_test)
  ```
- **常见应用场景**：房价预测、销量预测、趋势分析
- **注意事项**：假设线性关系，对异常值敏感，需检查多重共线性

### 1.2 逻辑回归
- **定义**：用于二分类问题的线性模型，输出概率值
- **数学原理**：P(y=1|x) = 1/(1 + e^{-(w·x+b)})，使用交叉熵损失
- **Python实现要点**：
  ```python
  from sklearn.linear_model import LogisticRegression
  model = LogisticRegression()
  model.fit(X_train, y_train)
  probabilities = model.predict_proba(X_test)
  ```
- **常见应用场景**：垃圾邮件检测、信用风险评估、疾病诊断
- **注意事项**：决策边界是线性的，需特征缩放，可扩展至多分类

### 1.3 决策树
- **定义**：基于特征阈值递归分割数据的树状模型
- **数学原理**：信息增益（ID3）、基尼不纯度（CART）、增益比（C4.5）
- **Python实现要点**：
  ```python
  from sklearn.tree import DecisionTreeClassifier
  model = DecisionTreeClassifier(max_depth=5)
  model.fit(X_train, y_train)
  ```
- **常见应用场景**：客户分群、医疗诊断、推荐系统
- **注意事项**：容易过拟合，需剪枝，对数据旋转敏感

### 1.4 随机森林
- **定义**：多棵决策树集成的Bagging方法
- **数学原理**：通过自助采样构建多个基学习器，投票或平均输出
- **Python实现要点**：
  ```python
  from sklearn.ensemble import RandomForestClassifier
  model = RandomForestClassifier(n_estimators=100)
  model.fit(X_train, y_train)
  ```
- **常见应用场景**：高维数据分类、特征重要性评估、异常检测
- **注意事项**：计算成本较高，可解释性较差，但泛化能力强

### 1.5 支持向量机
- **定义**：寻找最大间隔超平面的分类器
- **数学原理**：最小化 ||w||²，满足 y_i(w·x_i + b) ≥ 1，通过核技巧处理非线性
- **Python实现要点**：
  ```python
  from sklearn.svm import SVC
  model = SVC(kernel='rbf', C=1.0)
  model.fit(X_train, y_train)
  ```
- **常见应用场景**：文本分类、图像识别、生物信息学
- **注意事项**：对大规模数据训练慢，参数调优关键，内存占用高

## 二、优化算法与损失函数

### 2.1 梯度下降
- **定义**：通过梯度反方向迭代更新参数的优化方法
- **数学原理**：w_{t+1} = w_t - η∇L(w_t)，其中η为学习率
- **Python实现要点**：
  ```python
  def gradient_descent(X, y, learning_rate=0.01, epochs=1000):
      w = np.zeros(X.shape[1])
      for _ in range(epochs):
          gradient = 2/X.shape[0] * X.T @ (X @ w - y)
          w -= learning_rate * gradient
      return w
  ```
- **常见应用场景**：神经网络训练、线性模型优化
- **注意事项**：学习率选择关键，可能陷入局部最优，需监控收敛

### 2.2 Adam优化器
- **定义**：结合动量与自适应学习率的优化算法
- **数学原理**：维护一阶矩m_t和二阶矩v_t的指数移动平均，参数更新考虑偏差校正
- **Python实现要点**：
  ```python
  import torch.optim as optim
  optimizer = optim.Adam(model.parameters(), lr=0.001)
  ```
- **常见应用场景**：深度学习模型训练，尤其是大规模参数优化
- **注意事项**：默认参数通常有效，但对某些问题可能需要调整β₁、β₂

### 2.3 交叉熵损失
- **定义**：衡量预测概率分布与真实分布差异的损失函数
- **数学原理**：L = -∑_{i=1}^n y_i log(ŷ_i)，其中y_i为真实标签，ŷ_i为预测概率
- **Python实现要点**：
  ```python
  import torch.nn as nn
  criterion = nn.CrossEntropyLoss()
  loss = criterion(outputs, labels)
  ```
- **常见应用场景**：分类任务，尤其是多分类问题
- **注意事项**：需配合Softmax使用，对类别不平衡敏感

### 2.4 均方误差
- **定义**：回归任务中预测值与真实值平方差的平均值
- **数学原理**：MSE = (1/n)∑(y_i - ŷ_i)²
- **Python实现要点**：
  ```python
  from sklearn.metrics import mean_squared_error
  mse = mean_squared_error(y_true, y_pred)
  ```
- **常见应用场景**：回归模型评估，线性回归优化目标
- **注意事项**：对异常值敏感，单位与原始数据平方相同

## 三、模型评估指标

### 3.1 准确率
- **定义**：分类正确的样本占总样本的比例
- **数学原理**：Accuracy = (TP+TN)/(TP+TN+FP+FN)
- **Python实现要点**：
  ```python
  from sklearn.metrics import accuracy_score
  acc = accuracy_score(y_true, y_pred)
  ```
- **常见应用场景**：类别平衡的二分类或多分类任务
- **注意事项**：类别不平衡时可能产生误导，需结合其他指标

### 3.2 精确率与召回率
- **定义**：精确率衡量预测为正的样本中实际为正的比例；召回率衡量实际为正的样本中被预测为正的比例
- **数学原理**：Precision = TP/(TP+FP)，Recall = TP/(TP+FN)
- **Python实现要点**：
  ```python
  from sklearn.metrics import precision_score, recall_score
  precision = precision_score(y_true, y_pred)
  recall = recall_score(y_true, y_pred)
  ```
- **常见应用场景**：信息检索、医学诊断、欺诈检测
- **注意事项**：两者通常存在权衡，F1分数可平衡二者

### 3.3 ROC曲线与AUC
- **定义**：反映分类器在不同阈值下真正例率与假正例率关系的曲线，AUC为曲线下面积
- **数学原理**：TPR = TP/(TP+FN)，FPR = FP/(FP+TN)
- **Python实现要点**：
  ```python
  from sklearn.metrics import roc_curve, auc
  fpr, tpr, thresholds = roc_curve(y_true, y_scores)
  roc_auc = auc(fpr, tpr)
  ```
- **常见应用场景**：二分类器性能比较，阈值选择
- **注意事项**：AUC对类别分布不敏感，但计算成本较高

### 3.4 R²决定系数
- **定义**：回归模型中解释变量对因变量变异的解释比例
- **数学原理**：R² = 1 - SSE/SST，其中SSE为残差平方和，SST为总平方和
- **Python实现要点**：
  ```python
  from sklearn.metrics import r2_score
  r2 = r2_score(y_true, y_pred)
  ```
- **常见应用场景**：回归模型拟合优度评估
- **注意事项**：随变量增加而增加，调整R²可惩罚变量数

## 四、特征工程技术

### 4.1 标准化
- **定义**：将特征缩放到均值为0、方差为1的分布
- **数学原理**：z = (x - μ)/σ，其中μ为均值，σ为标准差
- **Python实现要点**：
  ```python
  from sklearn.preprocessing import StandardScaler
  scaler = StandardScaler()
  X_scaled = scaler.fit_transform(X)
  ```
- **常见应用场景**：基于距离的算法（如SVM、KNN）、梯度下降优化
- **注意事项**：对异常值敏感，鲁棒缩放可作为替代

### 4.2 独热编码
- **定义**：将分类变量转换为二进制向量表示
- **数学原理**：每个类别对应一个二进制特征，该类别出现则为1，否则为0
- **Python实现要点**：
  ```python
  from sklearn.preprocessing import OneHotEncoder
  encoder = OneHotEncoder()
  X_encoded = encoder.fit_transform(X_categorical)
  ```
- **常见应用场景**：树模型以外的机器学习算法
- **注意事项**：可能导致特征维度爆炸，需考虑稀疏表示

### 4.3 主成分分析
- **定义**：通过正交变换将数据投影到低维空间，保留最大方差
- **数学原理**：求解协方差矩阵的特征值与特征向量，选择前k个主成分
- **Python实现要点**：
  ```python
  from sklearn.decomposition import PCA
  pca = PCA(n_components=2)
  X_pca = pca.fit_transform(X)
  ```
- **常见应用场景**：数据可视化、噪声过滤、特征降维
- **注意事项**：线性假设，解释性下降，需标准化预处理

### 4.4 递归特征消除
- **定义**：通过递归减少特征集来选择最重要特征的包装方法
- **数学原理**：训练模型→计算特征重要性→移除最不重要特征→重复
- **Python实现要点**：
  ```python
  from sklearn.feature_selection import RFE
  selector = RFE(estimator=model, n_features_to_select=10)
  X_selected = selector.fit_transform(X, y)
  ```
- **常见应用场景**：高维数据特征选择，模型简化
- **注意事项**：计算成本高，依赖于基模型的准确性

## 五、算法复杂度分析

### 5.1 时间复杂度
- **定义**：算法执行时间随输入规模增长的变化趋势
- **数学原理**：使用大O表示法描述最坏情况下的渐近上界
- **Python实现要点**：分析循环嵌套、递归调用、数据结构操作
- **常见复杂度类**：
  - O(1)：常数时间，如数组索引
  - O(log n)：对数时间，如二分查找
  - O(n)：线性时间，如遍历数组
  - O(n log n)：线性对数时间，如快速排序
  - O(n²)：平方时间，如嵌套循环
- **注意事项**：关注最坏情况，但平均情况也可能重要

### 5.2 空间复杂度
- **定义**：算法执行过程中所需存储空间随输入规模的变化
- **数学原理**：考虑变量、数据结构、递归调用栈等内存占用
- **Python实现要点**：注意列表扩展、字典扩容、递归深度
- **常见场景**：
  - 原地算法：O(1)额外空间
  - 线性空间：O(n)额外空间
  - 递归算法：O(n)调用栈空间
- **注意事项**：Python内存管理开销，注意大对象复制

### 5.3 动态规划
- **定义**：通过将问题分解为重叠子问题并存储解来优化计算
- **数学原理**：最优子结构+重叠子问题，通过状态转移方程求解
- **Python实现要点**：
  ```python
  def fibonacci(n):
      dp = [0] * (n+1)
      dp[1] = 1
      for i in range(2, n+1):
          dp[i] = dp[i-1] + dp[i-2]
      return dp[n]
  ```
- **常见应用场景**：最短路径、背包问题、序列比对
- **注意事项**：确定状态定义，设计转移方程，注意边界条件

- **Python示例**：
  ```python

  def objective(trial):
      param = trial.suggest_float('learning_rate', 1e-5, 1e-2, log=True)
      # 训练并评估模型
      return accuracy
  study = optuna.create_study(direction='maximize')
  study.optimize(objective, n_trials=100)
  ```

- **定义**：自动化特征工程库
- **核心原理**：基于深度特征合成（DFS）自动生成特征
- **使用场景**：结构化数据特征工程，时序数据特征提取

