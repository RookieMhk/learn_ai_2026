### 二、Week 3 每日详细计划

#### **Day 15（深度学习Day 1）：神经网络数学基础**
**主题**：从线性回归到神经网络的数学跃迁
**时间**：2小时
**核心知识点**：
1. 生物神经元与人工神经元的类比
2. 感知机模型与前向传播计算
3. 激活函数（Sigmoid, ReLU, Tanh）的数学性质与应用场景
4. 多层感知机（MLP）的层次结构与表达能力

**交付物要求**：
1. **深度文档**：《神经网络前向传播数学详解.md》- 包含矩阵形式推导、激活函数导数、计算图表示
2. **代码实践**：《手动实现前向传播.py》- 不使用框架，纯NumPy实现3层MLP前向计算
3. **知识测验**：5道选择题，涵盖激活函数选择、梯度计算、层数设计原则

**具体任务**：
- 阅读神经网络数学原理，理解权重矩阵W、偏置向量b的维度关系
- 推导Sigmoid、ReLU的导数公式，理解梯度消失/爆炸问题
- 编写NumPy代码实现3层MLP前向传播，支持批量输入
- 测试不同激活函数对输出分布的影响

**关联Week 2**：延续线性回归的权重优化思维，扩展到多层非线性变换

#### **Day 16（深度学习Day 2）：反向传播与梯度下降优化**
**主题**：理解深度学习训练的“引擎原理”
**时间**：2小时
**核心知识点**：
1. 链式法则在神经网络中的具体应用
2. 损失函数对每层参数的梯度计算
3. 优化器（SGD, Adam, RMSProp）的数学原理与实现
4. 学习率调度策略（StepLR, CosineAnnealing）

**交付物要求**：
1. **深度文档**：《反向传播完整推导与优化器比较.md》- 从链式法则到Adam更新公式
2. **代码实践**：《手动实现反向传播.py》- 为Day15的MLP添加梯度计算与参数更新
3. **可视化工具**：《梯度流动可视化.ipynb》- 展示各层梯度范数变化，检测梯度问题

**具体任务**：
- 推导均方误差、交叉熵损失对网络输出的梯度
- 实现反向传播算法，计算W、b的梯度
- 对比SGD、Momentum、Adam在简单任务上的收敛速度
- 可视化训练过程中各层梯度的分布与变化

**关联最新趋势**：结合2026年注意力残差（Attention Residuals）架构，理解梯度流动优化

#### **Day 17（深度学习Day 3）：PyTorch核心API入门**
**主题**：掌握现代深度学习框架的基础操作
**时间**：1.5小时
**核心知识点**：
1. PyTorch张量（Tensor）创建与操作（CPU/GPU）
2. 自动微分（autograd）机制与计算图
3. 神经网络模块（nn.Module）的构建模式
4. 数据加载与预处理（DataLoader, Dataset）

**交付物要求**：
1. **代码实践**：《PyTorch基础操作练习.py》- 涵盖张量创建、索引、广播、矩阵运算
2. **项目模板**：《PyTorch训练流程模板.py》- 标准化的数据加载、模型定义、训练循环
3. **深度文档**：《PyTorch自动微分原理与应用指南.md》- 详细解释requires_grad、backward、grad

**具体任务**：
- 熟悉PyTorch张量API，实现常用数学运算
- 构建简单线性回归模型，理解autograd工作流程
- 创建自定义Dataset类，实现批量数据加载
- 编写完整的训练-验证-测试流程模板

**关联框架更新**：了解PyTorch 2.10的torch.compile、varlen_attn等新特性

#### **Day 18（深度学习Day 4）：TensorFlow核心API入门**

**交付物要求**：

2. **深度文档**：《TensorFlow 2.21新特性解析.md》- LiteRT、NPU加速、跨框架部署
3. **性能对比**：《框架性能基准测试.ipynb》- 在相同硬件上比较训练速度

**具体任务**：
- 学习TensorFlow 2.x的Eager Execution模式
- 使用Keras API构建CNN分类模型
- 掌握tf.data管道构建与性能优化技巧
- 对比PyTorch与TensorFlow在易用性、性能方面的差异

**关联最新趋势**：了解Google LiteRT边缘推理框架，掌握端侧部署趋势
