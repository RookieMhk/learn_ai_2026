# 特征工程实战指南

## 1. 特征工程概述

特征工程是机器学习流程中至关重要的一环，指通过对原始数据进行转换、组合、提取等操作，构造出对模型预测更有帮助的特征的过程。特征工程的质量直接决定模型性能的上限。

**核心目标**：
1. **提升模型性能**：构造更有区分度的特征，提升模型准确率
2. **降低计算复杂度**：通过特征选择减少特征维度，提高训练和推理速度
3. **增强模型可解释性**：构造符合业务逻辑的特征，便于结果分析

**工作流程**：
```
原始数据 → 数据预处理 → 特征构造 → 特征选择 → 特征提取 → 模型训练
```

## 2. 特征选择

特征选择旨在从原始特征集中选择出对目标变量预测最有用的特征子集，减少特征维度，防止过拟合。

### 2.1 过滤法（Filter Methods）

基于特征的统计属性进行筛选，独立于机器学习模型。

#### 2.1.1 方差选择法
**原理**：移除方差低于阈值的特征，认为低方差特征对目标区分度小。

**数学公式**：
- 特征 $Xⱼ$ 的方差：$Var(Xⱼ) = 1/(n-1) Σ_ᵢ₌₁ⁿ (x_ᵢⱼ - {X}ⱼ)²$
- 选择条件：$Var(Xⱼ) > threshold$

**适用场景**：数据预处理阶段快速剔除低信息量特征。

**实现示例**：
```python
from sklearn.feature_selection import VarianceThreshold

# 移除方差小于0.1的特征
selector = VarianceThreshold(threshold=0.1)
X_selected = selector.fit_transform(X)
```

#### 2.1.2 相关系数法
**原理**：计算特征与目标变量的相关系数，选择相关性强的特征。

**数学公式**：
- 皮尔逊相关系数：$r = {Σ_ᵢ₌₁ⁿ(xᵢ - {x})(yᵢ - {y})}{√Σ_ᵢ₌₁ⁿ(xᵢ - {x)² Σ_ᵢ₌₁ⁿ(yᵢ - {y})²}}$
- 选择条件：$|r| > threshold$

**适用场景**：线性关系较强的回归问题。

**实现示例**：
```python
import numpy as np
from scipy.stats import pearsonr

# 计算每个特征与目标的相关系数
correlations = [pearsonr(X[:, i], y)[0] for i in range(X.shape[1])]
# 选择相关系数绝对值大于0.3的特征
selected_indices = [i for i, corr in enumerate(correlations) if abs(corr) > 0.3]
X_selected = X[:, selected_indices]
```

#### 2.1.3 卡方检验
**原理**：检验特征与目标变量之间的独立性，适用于分类问题。

**数学公式**：
- 卡方统计量：$χ² = Σ ((O_ᵢⱼ - E_ᵢⱼ)²)/E_ᵢⱼ$
- 其中 $O_ᵢⱼ$ 为观察频数，$E_ᵢⱼ$ 为期望频数

**适用场景**：分类问题中的离散特征筛选。

**实现示例**：
```python
from sklearn.feature_selection import chi2, SelectKBest

# 选择与目标变量最相关的k个特征
selector = SelectKBest(chi2, k=10)
X_selected = selector.fit_transform(X, y)
```

### 2.2 包裹法（Wrapper Methods）

基于特定机器学习模型的性能评估来选择特征子集。

#### 2.2.1 递归特征消除（RFE）
**原理**：递归地移除最不重要的特征，基于模型权重或特征重要性。

**算法步骤**：
1. 用所有特征训练模型
2. 根据特征重要性排序
3. 移除最不重要的特征
4. 重复1-3直到达到指定特征数

**适用场景**：特征维度较高，需要精确控制特征数量的场景。

**实现示例**：
```python
from sklearn.feature_selection import RFE
from sklearn.linear_model import LogisticRegression

# 使用逻辑回归作为基模型，选择5个特征
estimator = LogisticRegression(max_iter=1000)
selector = RFE(estimator, n_features_to_select=5)
X_selected = selector.fit_transform(X, y)
```

#### 2.2.2 前向选择
**原理**：从空特征集开始，逐步添加最能提升模型性能的特征。

**算法步骤**：
1. 初始化空特征集 $S = ∅$
2. 对每个未选特征 $fᵢ ∉ S$，评估 $S ∪ {fᵢ}$ 的模型性能
3. 选择性能提升最大的特征加入 $S$
4. 重复2-3直到满足停止条件

**适用场景**：特征数量适中，计算资源充足的场景。

#### 2.2.3 后向消除
**原理**：从全特征集开始，逐步移除最不重要的特征。

**算法步骤**：
1. 初始化特征集 $S$ 包含所有特征
2. 对每个已选特征 $fᵢ ∈ S$，评估 $S {fᵢ}$ 的模型性能
3. 移除对性能影响最小的特征
4. 重复2-3直到满足停止条件

**适用场景**：特征数量适中，需要精简特征集的场景。

### 2.3 嵌入法（Embedded Methods）

特征选择过程嵌入到模型训练中，模型训练过程自动进行特征选择。

#### 2.3.1 Lasso回归
**原理**：在损失函数中加入L1正则化项，使部分特征的系数压缩为0。

**数学公式**：
- Lasso损失函数：$L(β) = 1/2n Σ_ᵢ₌₁ⁿ (yᵢ - Σ_ⱼ₌₁ᵖ x_ᵢⱼβⱼ)² + λ Σ_ⱼ₌₁ᵖ |βⱼ|$
- 其中 $λ$ 为正则化强度参数

**适用场景**：线性回归问题，需要自动特征选择的场景。

**实现示例**：
```python
from sklearn.linear_model import Lasso

# 使用Lasso进行特征选择
lasso = Lasso(alpha=0.01)
lasso.fit(X, y)
# 选择系数非零的特征
selected_indices = np.where(lasso.coef_ != 0)[0]
X_selected = X[:, selected_indices]
```

#### 2.3.2 树模型特征重要性
**原理**：基于树模型（如随机森林、梯度提升树）计算的特征重要性。

**计算方法**：
- 基尼重要性：特征在分裂节点时减少的基尼不纯度总和
- 信息增益：特征在分裂节点时增加的信息增益总和

**适用场景**：非线性问题，需要评估特征重要性的场景。

**实现示例**：
```python
from sklearn.ensemble import RandomForestClassifier

# 使用随机森林计算特征重要性
rf = RandomForestClassifier(n_estimators=100)
rf.fit(X, y)
# 获取特征重要性
importances = rf.feature_importances_
# 选择重要性大于阈值的特征
selected_indices = np.where(importances > 0.01)[0]
X_selected = X[:, selected_indices]
```

#### 2.3.3 正则化逻辑回归
**原理**：在逻辑回归中加入L1或L2正则化，实现特征选择。

**数学公式**：
- L1正则化逻辑回归损失函数：
$L(β) = -1/n Σ_ᵢ₌₁ⁿ [yᵢ (σ(βᵀ xᵢ)) + (1-yᵢ) (1-σ(βᵀ xᵢ))] + λ |β|₁$
- 其中 $σ(z) = 1/(1+e⁻ᶻ)$ 为sigmoid函数

**适用场景**：分类问题，需要稀疏特征选择的场景。

## 3. 特征提取

特征提取通过线性或非线性变换将原始特征空间映射到低维空间，提取数据的内在结构。

### 3.1 主成分分析（PCA）

**原理**：通过正交变换将原始特征转换为一组线性不相关的主成分，按方差大小排序。

**数学基础**：
1. 计算协方差矩阵：$C = 1/(n-1) Xᵀ X$
2. 特征值分解：$C = V Λ Vᵀ$
3. 选择前k个特征值对应的特征向量作为投影矩阵

**方差解释率**：第i个主成分解释的方差比例为 $λᵢ/Σ_ⱼ₌₁ᵖ λⱼ$

**适用场景**：高维数据可视化、数据压缩、去除特征相关性。

**实现示例**：
```python
from sklearn.decomposition import PCA

# 保留95%的方差信息
pca = PCA(n_components=0.95)
X_pca = pca.fit_transform(X)
print(f"原始特征数: {X.shape[1]}, PCA后特征数: {X_pca.shape[1]}")
print(f"解释方差比例: {sum(pca.explained_variance_ratio_):.3f}")
```

### 3.2 线性判别分析（LDA）

**原理**：寻找使类间方差最大、类内方差最小的投影方向，适用于有监督的降维。

**数学基础**：
1. 计算类内散度矩阵：$SW = Σ_ᵢ₌₁ᶜ Σ_ₓ ᵢₙ C_ᵢ (x - μᵢ)(x - μᵢ)ᵀ$
2. 计算类间散度矩阵：$SB = Σ_ᵢ₌₁ᶜ nᵢ (μᵢ - μ)(μᵢ - μ)ᵀ$
3. 求解广义特征值问题：$SB v = λ SW v$

**优化目标**：最大化Fisher准则 $J(v) = (vᵀ SB v)/(vᵀ SW v)$

**适用场景**：分类问题的特征降维，类别数较少的情况。

**实现示例**：
```python
from sklearn.discriminant_analysis import LinearDiscriminantAnalysis

# LDA降维（最多降至类别数-1维）
lda = LinearDiscriminantAnalysis(n_components=2)
X_lda = lda.fit_transform(X, y)
```

### 3.3 t-SNE（t-Distributed Stochastic Neighbor Embedding）

**原理**：基于概率分布的非线性降维方法，保持高维空间中样本间的相对距离。

**数学基础**：
1. 计算高维空间相似度：$p_ⱼ|ᵢ = ((-|xᵢ - xⱼ|² / 2σᵢ²))/(Σ_ₖ ₙₑq ᵢ (-|xᵢ - xₖ|² / 2σᵢ²))$
2. 定义低维空间相似度：$q_ᵢⱼ = ((1 + |yᵢ - yⱼ|²)⁻¹)/(Σ_ₖ ₙₑq ₗ (1 + |yₖ - yₗ|²)⁻¹)$
3. 最小化KL散度：$C = KL(P|Q) = Σᵢ Σⱼ p_ᵢⱼ p_ᵢⱼ/q_ᵢⱼ$

**特点**：擅长捕捉局部结构，适合可视化但计算复杂度高。

**适用场景**：高维数据可视化，探索数据聚类结构。

**实现示例**：
```python
from sklearn.manifold import TSNE

# t-SNE降维可视化
tsne = TSNE(n_components=2, perplexity=30, random_state=42)
X_tsne = tsne.fit_transform(X)
```

## 4. 特征构造

特征构造通过组合、变换现有特征，生成新的更有信息量的特征。

### 4.1 多项式特征

**原理**：通过特征的多项式组合（如平方、交叉项）引入非线性关系。

**数学公式**：
- 二阶多项式特征：$φ(x) = [1, x₁, x₂, x₁², x₁x₂, x₂²]$
- 一般形式：$φ(x) = { ∏_ⱼ₌₁ᵖ xⱼᵈⱼ Σ_ⱼ₌₁ᵖ dⱼ ≤ d }$

**适用场景**：线性模型引入非线性能力，回归问题中特征关系复杂的情况。

**实现示例**：
```python
from sklearn.preprocessing import PolynomialFeatures

# 生成2阶多项式特征
poly = PolynomialFeatures(degree=2, include_bias=False)
X_poly = poly.fit_transform(X)
print(f"原始特征数: {X.shape[1]}, 多项式特征数: {X_poly.shape[1]}")
```

### 4.2 交互特征

**原理**：将两个或多个特征相乘或相除，捕捉特征间的交互效应。

**常见交互方式**：
1. 乘积交互：$x_ₙₑw = xᵢ × xⱼ$
2. 比值交互：$x_ₙₑw = xᵢ / xⱼ$（需避免除零）
3. 差值交互：$x_ₙₑw = |xᵢ - xⱼ|$

**适用场景**：特征间存在显著交互效应，如用户行为分析、金融风控。

**实现示例**：
```python
import numpy as np

# 创建交互特征：特征乘积
X_interaction = X.copy()
# 假设X有特征0和1，创建它们的交互特征
X_interaction = np.column_stack([X_interaction, X[:, 0] * X[:, 1]])
# 创建比值特征（添加小常数避免除零）
X_interaction = np.column_stack([X_interaction, X[:, 0] / (X[:, 1] + 1e-8)])
```

### 4.3 时序特征

**原理**：针对时间序列数据，构造滑动窗口统计特征。

**常用时序特征**：
1. 滑动统计量：均值、标准差、最大值、最小值
2. 滞后特征：$x_ₜ₋ₗₐg$，捕捉历史信息
3. 差分特征：$xₜ - x_ₜ₋₁$，捕捉变化趋势
4. 周期特征：小时、星期、月份等时间成分

**数学公式**：
- 滑动窗口均值：${x}_ₜ,w = 1/w Σ_ᵢ₌ₜ₋w₊₁ᵗ xᵢ$
- 一阶差分：$Δ xₜ = xₜ - x_ₜ₋₁$

**适用场景**：时间序列预测、用户行为序列分析。

**实现示例**：
```python
import pandas as pd

# 创建时间序列DataFrame
df = pd.DataFrame({'value': X.flatten()})
# 添加滞后特征
df['lag_1'] = df['value'].shift(1)
df['lag_2'] = df['value'].shift(2)
# 添加滑动窗口统计
df['rolling_mean_3'] = df['value'].rolling(window=3).mean()
df['rolling_std_3'] = df['value'].rolling(window=3).std()
# 添加差分特征
df['diff_1'] = df['value'].diff(1)
```

## 5. 特征工程最佳实践

### 5.1 处理流程建议
1. **先清洗后构造**：确保数据质量后再进行特征工程
2. **分阶段验证**：每步特征工程后评估模型性能变化
3. **避免数据泄露**：特征工程应在训练集上进行，再应用到测试集

### 5.2 特征工程检查清单
- [ ] 缺失值处理是否恰当
- [ ] 异常值检测与处理
- [ ] 特征缩放是否必要
- [ ] 分类特征编码是否合理
- [ ] 特征选择是否去除冗余
- [ ] 构造特征是否具有业务意义

### 5.3 性能评估指标
- **模型性能**：准确率、AUC、RMSE等指标提升
- **计算效率**：训练和推理时间减少
- **可解释性**：特征重要性可解释，符合业务逻辑

## 6. 总结

特征工程是机器学习中兼具科学性与艺术性的环节。通过系统性地应用特征选择、特征提取和特征构造技术，可以显著提升模型性能。关键是要结合具体业务场景，理解数据背后的意义，构造既有统计显著性又有业务解释性的特征。

**核心要点**：
1. **特征选择**关注"选什么"，通过过滤、包裹、嵌入方法精简特征集
2. **特征提取**关注"怎么变"，通过线性或非线性变换提取数据本质结构
3. **特征构造**关注"怎么创"，通过组合、变换创造新的信息量特征

在实际应用中，建议采用迭代式方法：构建基础特征集→训练模型→分析错误→构造新特征→重新训练，持续优化特征工程效果。
