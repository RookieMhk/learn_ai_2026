### 1. 三个核心收获

**收获一：前向传播的矩阵运算本质**

```
今天学习到，神经网络的前向传播本质上是线性变换的叠加：
z = Wx + b
a = σ(z)

其中：
- W 是权重矩阵，形状为 (当前层神经元数, 上一层神经元数)
- b 是偏置向量，形状为 (当前层神经元数, 1)
- σ 是激活函数，引入非线性能力

通过矩阵运算，神经网络可以高效处理批量数据，计算复杂度为 O(n³)。
```

**收获二：反向传播的链式法则应用**

```
反向传播的核心是链式法则的逐层应用：
∂L/∂W = ∂L/∂a · ∂a/∂z · ∂z/∂W

对于输出层：
∂L/∂z² = a - y  (交叉熵损失+Sigmoid激活的简化形式)

梯度从输出层开始，通过链式法则逐层反向传播，最终计算各参数的梯度。
这种机制使得神经网络可以通过梯度下降进行高效训练。
```

**收获三：激活函数导数的数学特性**

```
不同激活函数的导数具有显著差异：
- Sigmoid: σ'(z) = σ(z)(1-σ(z))，梯度范围(0, 0.25]，存在梯度消失问题
- ReLU: ReLU'(z) = 1 if z>0 else 0，正区间梯度为1，缓解梯度消失
- Tanh: tanh'(z) = 1 - tanh²(z)，梯度范围(0, 1]，相比Sigmoid有所改善

理解这些导数特性有助于选择合适的激活函数和优化训练过程。
```
