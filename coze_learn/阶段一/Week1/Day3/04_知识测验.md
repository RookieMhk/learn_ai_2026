### 题目3：贝叶斯定理计算

## ✅ 正确答案与解析

### 题目1解析
**正确答案：D**

**解析：**
- A正确：正态分布的概率密度函数是以均值μ为中心的对称钟形曲线
- B正确：对于对称分布，均值、中位数和众数相等
- C正确：正态分布的峰度系数为3（有些定义中会减去3，称为超额峰度）
- D错误：方差越大，数据越分散，曲线越扁平；方差越小，曲线越尖峭

**知识要点：** 正态分布的方差与曲线形态的关系是反直觉的，需要特别注意。

---

### 题目2解析
**正确答案：A**

**解析：**
- 逻辑回归是典型的二分类模型
- sigmoid函数将线性输出映射到(0,1)区间，表示样本属于正类的概率
- 每个样本的分类结果（0或1）服从伯努利分布，成功概率为sigmoid输出
- 其他选项虽然相关，但不是最准确描述

**知识要点：** 理解逻辑回归输出的概率意义及其与伯努利分布的关系。

---

### 题目3解析
**正确答案：A**

**解析：**
根据贝叶斯定理：
```
P(病|阳) = P(阳|病) * P(病) / P(阳)
```
其中：
- P(阳) = P(阳|病)*P(病) + P(阳|健康)*P(健康)
- P(阳) = 0.95*0.01 + 0.05*0.99 = 0.0095 + 0.0495 = 0.059

计算后验概率：
```
P(病|阳) = 0.95 * 0.01 / 0.059 ≈ 0.161
```
约等于0.16。

**知识要点：** 贝叶斯定理在医学检测中的经典应用，低发病率下即使检测准确率高，后验概率也可能不高。

### 题目4解析
**正确答案：B**

**解析：**
数据：[2, 4, 6, 8, 10]
- 均值 = (2+4+6+8+10)/5 = 30/5 = 6
- 方差 = [(2-6)² + (4-6)² + (6-6)² + (8-6)² + (10-6)²]/5 = (16+4+0+4+16)/5 = 40/5 = 8
- 标准差 = √8 = 2√2 ≈ 2.828
- 中位数 = 6（中间值）

因此：
- A错误：方差为8，不是8（计算正确，但表述不一致）
- B正确：均值=6，标准差=2√2
- C错误：方差为8，不是10
- D错误：标准差为2√2，不是√10

**知识要点：** 区分均值、中位数、方差、标准差的计算。

---

### 题目5解析
**正确答案：A**

**解析：**
- A正确：皮尔逊相关系数|r|>0.7通常被认为强相关，0.85表示强正相关
- B错误：相关不等于因果，需要实验设计验证因果关系
- C错误：皮尔逊相关系数只度量线性关系强度，不证明因果关系
- D错误：相关系数高不代表必然性，可能存在其他影响因素

**知识要点：** 区分相关性、因果关系、必然性的概念，这是统计思维的核心。
