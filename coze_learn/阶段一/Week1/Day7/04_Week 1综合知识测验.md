## 第一部分：线性代数（5题）

### 1. 向量点积与余弦相似度
已知向量 **a** = [3, 4, 0]，向量 **b** = [1, 2, 2]，请计算：
1) **a** 与 **b** 的点积
2) **a** 的模长
3) **b** 的模长
4) 余弦相似度 cosθ

**选项**：
A) 点积=11, |a|=5, |b|=3, cosθ=11/15
B) 点积=11, |a|=5, |b|=3, cosθ=11/18
C) 点积=10, |a|=5, |b|=3, cosθ=10/15
D) 点积=10, |a|=5, |b|=3, cosθ=10/18

**正确答案**：A

**解析**：
- 点积：3×1 + 4×2 + 0×2 = 3 + 8 + 0 = 11
- |a| = √(3² + 4² + 0²) = √(9+16+0) = √25 = 5
- |b| = √(1² + 2² + 2²) = √(1+4+4) = √9 = 3
- cosθ = 点积/(|a|×|b|) = 11/(5×3) = 11/15

### 2. 矩阵乘法维度
矩阵 A 的维度为 (3, 4)，矩阵 B 的维度为 (4, 5)，矩阵 C 的维度为 (5, 2)。请问以下哪个矩阵乘法是合法的，且结果的维度是多少？

**选项**：
A) A × B × C，结果维度为 (3, 2)
B) B × C × A，结果维度为 (4, 2)
C) C × A × B，结果维度为 (2, 5)
D) A × C × B，结果维度为 (3, 5)

**正确答案**：A

**解析**：
矩阵乘法要求前一个矩阵的列数等于后一个矩阵的行数。
- A(3,4) × B(4,5) → 结果维度 (3,5)
- (3,5) × C(5,2) → 结果维度 (3,2)
其他选项不满足维度匹配条件。

### 3. 特征值与特征向量
对于矩阵 M = [[2, 1], [1, 2]]，以下哪组是其特征值和特征向量？

**选项**：
A) λ₁=3, v₁=[1,1]; λ₂=1, v₂=[1,-1]
B) λ₁=4, v₁=[1,1]; λ₂=0, v₂=[1,-1]
C) λ₁=3, v₁=[1,-1]; λ₂=1, v₂=[1,1]
D) λ₁=4, v₁=[1,-1]; λ₂=0, v₂=[1,1]

**正确答案**：A

**解析**：
计算特征值：det(M - λI) = 0
|2-λ, 1|
|1, 2-λ| = (2-λ)² - 1 = λ² - 4λ + 3 = 0
解得 λ₁=3, λ₂=1

对于 λ₁=3：解 (M-3I)v=0 → [[-1,1],[1,-1]]v=0 → v₁=[1,1]
对于 λ₂=1：解 (M-I)v=0 → [[1,1],[1,1]]v=0 → v₂=[1,-1]

### 4. 奇异值分解（SVD）应用
在图像压缩中，我们使用SVD对图像矩阵进行低秩近似。假设原始图像矩阵为 A（大小为 m×n），经过SVD得到 A = UΣVᵀ。如果我们只保留前 k 个奇异值，近似矩阵 Aₖ 的秩是多少？

**选项**：
A) k
B) min(m, n)
C) max(m, n)
D) m + n - k

**正确答案**：A

**解析**：
SVD中，Σ是对角矩阵，对角线元素为奇异值。只保留前k个奇异值意味着Σ中只有k个非零元素，因此近似矩阵 Aₖ = U(:,1:k) × Σ(1:k,1:k) × Vᵀ(1:k,:) 的秩为 k，因为这是三个矩阵乘积的秩，其中 Σ(1:k,1:k) 是满秩的 k×k 对角矩阵。

### 5. 矩阵的秩与线性相关性
已知矩阵 A = [[1,2,3], [2,4,6], [3,6,9]]，以下说法正确的是：

**选项**：
A) 矩阵的秩为3，三列线性无关
B) 矩阵的秩为2，第三列是第一列和第二列的线性组合
C) 矩阵的秩为1，三列成比例
D) 矩阵的秩为0，所有元素相同

**正确答案**：C

**解析**：
观察矩阵 A：
- 第二行是第一行的2倍：2×[1,2,3] = [2,4,6]
- 第三行是第一行的3倍：3×[1,2,3] = [3,6,9]
因此行向量线性相关，秩为1。实际上三列也是成比例的：第二列是第一列的2倍，第三列是第一列的3倍。

## 第二部分：概率统计（5题）

### 6. 条件概率与贝叶斯定理
某疾病的发病率为1%，检测方法的准确率为：患者检测阳性的概率为99%，健康人检测阳性的概率为5%。如果一个人检测结果为阳性，他实际患病的概率是多少？

**选项**：
A) 约 16.7%
B) 约 25.0%
C) 约 33.3%
D) 约 50.0%

**正确答案**：A

**解析**：
使用贝叶斯定理：
P(患病|阳性) = P(阳性|患病)×P(患病) / P(阳性)
P(阳性) = P(阳性|患病)×P(患病) + P(阳性|健康)×P(健康)
= 0.99×0.01 + 0.05×0.99 = 0.0099 + 0.0495 = 0.0594

P(患病|阳性) = 0.99×0.01 / 0.0594 ≈ 0.0099/0.0594 ≈ 0.1667 ≈ 16.7%

### 7. 期望值与方差
随机变量 X 的概率分布如下：
P(X=1) = 0.2, P(X=2) = 0.3, P(X=3) = 0.4, P(X=4) = 0.1
计算 E(X) 和 Var(X)。

**选项**：
A) E(X)=2.5, Var(X)=0.85
B) E(X)=2.5, Var(X)=0.95
C) E(X)=2.6, Var(X)=0.84
D) E(X)=2.6, Var(X)=0.94

**正确答案**：C

**解析**：
E(X) = 1×0.2 + 2×0.3 + 3×0.4 + 4×0.1 = 0.2 + 0.6 + 1.2 + 0.4 = 2.6

E(X²) = 1²×0.2 + 2²×0.3 + 3²×0.4 + 4²×0.1 = 1×0.2 + 4×0.3 + 9×0.4 + 16×0.1 
= 0.2 + 1.2 + 3.6 + 1.6 = 6.6

Var(X) = E(X²) - [E(X)]² = 6.6 - (2.6)² = 6.6 - 6.76 = 0.84

### 8. 正态分布与标准化
某考试成绩服从正态分布 N(75, 10²)。如果要计算成绩高于85分的概率，需要先进行标准化。85分对应的Z分数是多少？

**选项**：
A) 0.5
B) 1.0
C) 1.5
D) 2.0

**正确答案**：B

**解析**：
Z分数公式：Z = (X - μ) / σ
其中 μ=75, σ=10, X=85
Z = (85-75)/10 = 10/10 = 1.0

成绩高于85分的概率为 P(Z > 1.0) ≈ 0.1587

### 9. 协方差与相关系数
两个随机变量 X 和 Y 的协方差 Cov(X,Y) = 12，X 的标准差 σ_X = 4，Y 的标准差 σ_Y = 3。它们的相关系数 ρ 是多少？

**选项**：
A) 0.5
B) 0.75
C) 1.0
D) 1.5

**正确答案**：C

**解析**：
相关系数 ρ = Cov(X,Y) / (σ_X × σ_Y)
= 12 / (4 × 3) = 12 / 12 = 1.0

相关系数为1.0表示完全正相关，但这在实际数据中很少见，本题为理论计算。

### 10. 中心极限定理应用
某工厂生产的产品重量服从均值为100g，标准差为5g的分布。随机抽取36个产品，求样本平均重量在99g到101g之间的概率。

**选项**：
A) 约 68.3%
B) 约 76.4%
C) 约 84.1%
D) 约 95.4%

**正确答案**：B

**解析**：
根据中心极限定理，样本均值服从 N(μ, σ²/n)
其中 μ=100, σ=5, n=36
样本均值的标准差 = σ/√n = 5/√36 = 5/6 ≈ 0.8333

标准化：
Z₁ = (99-100)/0.8333 ≈ -1.2
Z₂ = (101-100)/0.8333 ≈ 1.2

P(-1.2 < Z < 1.2) = Φ(1.2) - Φ(-1.2) = 0.8849 - 0.1151 = 0.7698 ≈ 76.4%
