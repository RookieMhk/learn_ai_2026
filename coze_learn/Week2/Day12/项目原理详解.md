# 项目原理详解：机器学习算法数学基础

## 1. 线性回归（Linear Regression）

### 1.1 数学原理

线性回归假设目标变量 $y$ 与特征 $x_1, x_2, ..., x_n$ 之间存在线性关系：

$$y = w_0 + w_1x_1 + w_2x_2 + ... + w_nx_n + \epsilon$$

其中：
- $w_0$ 是截距（偏置项）
- $w_1, w_2, ..., w_n$ 是特征权重
- $\epsilon$ 是误差项，假设服从正态分布 $N(0, \sigma^2)$

### 1.2 最小二乘法推导

目标是最小化残差平方和（RSS）：

$$\text{RSS} = \sum_{i=1}^m (y_i - \hat{y}_i)^2 = \sum_{i=1}^m \left(y_i - (w_0 + \sum_{j=1}^n w_jx_{ij})\right)^2$$

矩阵形式：
$$\text{RSS} = (Y - XW)^T(Y - XW)$$

求导并令导数为零：
$$\frac{\partial \text{RSS}}{\partial W} = -2X^TY + 2X^TXW = 0$$

解得最优权重：
$$\hat{W} = (X^TX)^{-1}X^TY$$

### 1.3 几何解释

- **投影视角**：线性回归的预测值 $\hat{Y}$ 是 $Y$ 在特征空间 $X$ 的列空间上的正交投影
- **误差正交性**：残差向量 $Y - \hat{Y}$ 与所有特征向量正交

### 1.4 假设条件

1. **线性关系**：因变量与自变量之间存在线性关系
2. **独立性**：观测值之间相互独立
3. **同方差性**：误差项的方差恒定
4. **正态性**：误差项服从正态分布
5. **无多重共线性**：特征之间不存在高度相关性

### 1.5 优缺点

**优点**：
- 简单直观，计算效率高
- 可解释性强，权重有明确意义
- 理论基础完善

**缺点**：
- 对非线性关系建模能力有限
- 对异常值敏感
- 假设条件严格，现实中往往难以满足

## 2. 岭回归（Ridge Regression）

### 2.1 数学原理

岭回归在线性回归的基础上增加L2正则化项：

$$\text{RSS}_{ridge} = \sum_{i=1}^m (y_i - \hat{y}_i)^2 + \lambda \sum_{j=1}^n w_j^2$$

其中 $\lambda \geq 0$ 是正则化强度参数。

### 2.2 求解方法

$$\hat{W}_{ridge} = (X^TX + \lambda I)^{-1}X^TY$$

### 2.3 几何解释

- **约束优化**：在权重向量的L2范数球内寻找最小RSS解
- **收缩效应**：所有权重向零收缩，但不会完全为零

### 2.4 适用场景

- 特征数量多，样本数量少
- 特征之间存在多重共线性
- 防止过拟合，提高模型泛化能力

## 3. Lasso回归（Least Absolute Shrinkage and Selection Operator）

### 3.1 数学原理

Lasso回归使用L1正则化：

$$\text{RSS}_{lasso} = \sum_{i=1}^m (y_i - \hat{y}_i)^2 + \lambda \sum_{j=1}^n |w_j|$$

### 3.2 几何解释

- **稀疏解**：L1正则化倾向于产生稀疏权重向量（许多权重为零）
- **特征选择**：自动完成特征选择，只保留重要特征

### 3.3 求解方法

由于L1正则化的不可导性，常用坐标下降法或最小角回归（LARS）求解。

### 3.4 优缺点

**优点**：
- 自动特征选择
- 处理高维数据有效
- 可解释性更强（稀疏模型）

**缺点**：
- 当 $n > m$ 时最多选择 $m$ 个特征
- 如果特征高度相关，可能随机选择一个而忽略其他

## 4. 决策树回归（Decision Tree Regressor）

### 4.1 算法原理

决策树通过递归地将特征空间划分为矩形区域，使每个区域内的目标变量方差最小化。

### 4.2 分裂准则

对于回归问题，常用均方误差（MSE）作为分裂准则：

$$\text{MSE} = \frac{1}{N} \sum_{i=1}^N (y_i - \bar{y})^2$$

其中 $\bar{y}$ 是节点内样本的平均值。

分裂时选择使加权MSE减少最多的特征和阈值：

$$\text{Gain} = \text{MSE}_{\text{parent}} - \left(\frac{N_{\text{left}}}{N}\text{MSE}_{\text{left}} + \frac{N_{\text{right}}}{N}\text{MSE}_{\text{right}}\right)$$

### 4.3 停止条件

1. 节点样本数小于最小分裂样本数
2. 节点深度达到最大深度
3. MSE减少量小于阈值
4. 所有特征值相同

### 4.4 几何解释

- **分层划分**：特征空间被划分为多个超矩形
- **分段常数**：每个区域内的预测值是常数（区域平均值）

### 4.5 优缺点

**优点**：
- 可解释性强，可视化直观
- 不需要特征缩放
- 能处理非线性关系
- 对异常值不敏感

**缺点**：
- 容易过拟合，需要剪枝
- 不稳定，数据微小变化可能导致完全不同的树
- 预测表面不平滑（阶梯状）

## 5. 随机森林回归（Random Forest Regressor）

### 5.1 集成学习原理

随机森林通过构建多个决策树并集成它们的预测：

$$\hat{y}_{RF} = \frac{1}{T} \sum_{t=1}^T \hat{y}_t$$

其中 $T$ 是树的数量，$\hat{y}_t$ 是第 $t$ 棵树的预测。

### 5.2 Bagging（Bootstrap Aggregating）

1. 从训练集中有放回地随机抽取 $m$ 个样本（自助采样）
2. 对每个自助样本训练一棵决策树
3. 重复 $T$ 次，得到 $T$ 棵决策树

### 5.3 随机特征选择

在每个节点分裂时，随机选择 $k$ 个特征（通常 $k = \sqrt{n}$）作为候选特征，增加树之间的多样性。

### 5.4 数学性质

**方差减少**：Bagging能有效减少模型方差，而不显著增加偏差。

**泛化误差界**：对于随机森林，泛化误差满足：

$$PE^* \leq \bar{\rho}(1 - s^2)/s^2$$

其中：
- $\bar{\rho}$ 是树之间的平均相关系数
- $s$ 是树的平均强度

### 5.5 优缺点

**优点**：
- 高准确率，泛化能力强
- 能处理高维数据，自动特征选择
- 对异常值和噪声鲁棒
- 提供特征重要性评估

**缺点**：
- 模型复杂度高，训练和预测速度慢
- 可解释性较差（黑箱模型）
- 需要调参（树数量、深度等）

## 6. 梯度提升回归（Gradient Boosting Regressor）

### 6.1 Boosting原理

梯度提升通过逐步添加弱学习器（通常是浅层决策树）来修正前序模型的残差：

$$F_m(x) = F_{m-1}(x) + \gamma_m h_m(x)$$

其中 $h_m(x)$ 是第 $m$ 个弱学习器，$\gamma_m$ 是学习率。

### 6.2 梯度下降视角

将损失函数 $L(y, F(x))$ 的优化视为函数空间中的梯度下降：

$$F_m(x) = F_{m-1}(x) - \gamma_m \nabla_{F_{m-1}} L(y, F_{m-1}(x))$$

### 6.3 算法步骤

对于均方误差损失 $L(y, F) = \frac{1}{2}(y - F)^2$：

1. 初始化：$F_0(x) = \bar{y}$
2. 对于 $m = 1$ 到 $M$：
   a. 计算伪残差：$r_{im} = y_i - F_{m-1}(x_i)$
   b. 拟合弱学习器 $h_m(x)$ 到伪残差 $(x_i, r_{im})$
   c. 更新：$F_m(x) = F_{m-1}(x) + \gamma h_m(x)$

### 6.4 正则化技术

1. **学习率收缩**：使用较小的学习率 $\gamma$
2. **子采样**：每次迭代随机选择部分样本
3. **早停法**：监控验证集误差，提前停止

### 6.5 优缺点

**优点**：
- 高预测精度，竞赛常用
- 灵活处理各种类型的数据
- 提供特征重要性

**缺点**：
- 训练时间长，计算资源要求高
- 容易过拟合，需要仔细调参
- 对异常值敏感

## 7. 模型选择与比较

### 7.1 算法复杂度对比

| 算法 | 训练复杂度 | 预测复杂度 | 内存需求 |
|------|------------|------------|----------|
| 线性回归 | $O(n^3)$ | $O(n)$ | $O(n^2)$ |
| 决策树 | $O(mn\log n)$ | $O(\log n)$ | $O(n)$ |
| 随机森林 | $O(Tmn\log n)$ | $O(T\log n)$ | $O(Tn)$ |
| 梯度提升 | $O(Tmn\log n)$ | $O(T\log n)$ | $O(Tn)$ |

### 7.2 适用场景总结

1. **线性关系明确，特征少**：线性回归
2. **特征多，有共线性**：岭回归
3. **特征选择需求强**：Lasso回归
4. **可解释性优先**：决策树
5. **高精度，稳定预测**：随机森林
6. **竞赛级别精度**：梯度提升

### 7.3 实践建议

1. 从简单模型开始（线性回归），建立基线
2. 检查线性假设是否成立（残差分析）
3. 根据数据特点选择合适的正则化方法
4. 使用集成方法提升性能，但要注意计算成本
5. 始终通过交叉验证评估泛化能力

## 8. 数学附录

### 8.1 矩阵求导公式

$$\frac{\partial}{\partial W} (Y - XW)^T(Y - XW) = -2X^TY + 2X^TXW$$

### 8.2 正则化项性质

- **L1正则化**：产生稀疏解，$|w|$ 在 $w=0$ 处不可导
- **L2正则化**：产生稠密解，$w^2$ 处处可导

### 8.3 偏差-方差分解

期望泛化误差可分解为：

$$\mathbb{E}[(y - \hat{f})^2] = \text{Bias}^2(\hat{f}) + \text{Var}(\hat{f}) + \sigma^2$$

其中：
- $\text{Bias}^2 = \mathbb{E}[\hat{f}] - f$
- $\text{Var} = \mathbb{E}[(\hat{f} - \mathbb{E}[\hat{f}])^2]$
- $\sigma^2$ 是数据本身的噪声
