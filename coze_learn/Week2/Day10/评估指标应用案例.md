## 案例一：二分类问题 - 信用评分模型

### 1.1 业务背景
**场景描述**：银行需要构建信用评分模型，预测客户是否会违约（二分类问题）。
- **正类**：违约（1）
- **负类**：正常还款（0）

**业务特点**：
- **代价不平衡**：误将高风险客户判为低风险（FN）的损失 > 误将低风险判为高风险（FP）的损失
- **样本不平衡**：违约客户通常只占5-10%

### 1.2 数据集与模型准备

```python
import numpy as np
import pandas as pd
from sklearn.model_selection import train_test_split
from sklearn.ensemble import RandomForestClassifier
from sklearn.datasets import make_classification
import matplotlib.pyplot as plt
plt.rcParams['font.sans-serif'] = ['Noto Sans CJK JP']
plt.rcParams['axes.unicode_minus'] = False

# 生成模拟信用评分数据
def generate_credit_data(n_samples=1000, random_state=42):
    """
    生成模拟信用评分数据集
    特征：年龄、收入、负债比、信用卡数量、历史逾期次数
    标签：0（正常），1（违约）
    """
    X, y = make_classification(
        n_samples=n_samples,
        n_features=5,
        n_informative=4,
        n_redundant=1,
        n_clusters_per_class=2,
        weights=[0.9, 0.1],  # 10%违约
        random_state=random_state
    )
    
    # 添加特征名称
    feature_names = ['年龄标准化', '收入标准化', '负债比', '信用卡数量', '历史逾期次数']
    
    # 转换为DataFrame方便分析
    df = pd.DataFrame(X, columns=feature_names)
    df['违约标签'] = y
    
    return df, X, y, feature_names

# 生成数据并划分训练集/测试集
df, X, y, feature_names = generate_credit_data()
X_train, X_test, y_train, y_test = train_test_split(
    X, y, test_size=0.3, random_state=42, stratify=y
)

print(f"数据集大小: {df.shape}")
print(f"违约比例: {df['违约标签'].mean():.1%}")
print(f"训练集: {X_train.shape[0]} 样本, 测试集: {X_test.shape[0]} 样本")
```

### 1.3 模型训练与预测

```python
# 训练随机森林分类器
model = RandomForestClassifier(
    n_estimators=100,
    max_depth=10,
    random_state=42,
    class_weight='balanced'  # 处理类别不平衡
)
model.fit(X_train, y_train)

# 预测概率和标签
y_pred_proba = model.predict_proba(X_test)[:, 1]  # 违约概率
y_pred = (y_pred_proba > 0.5).astype(int)  # 阈值0.5的预测标签
```

## 案例二：回归问题 - 房价预测模型

### 2.1 业务背景
**场景描述**：房地产平台需要预测房屋售价，辅助定价和投资决策。

**业务特点**：
- **连续值预测**：输出为连续数值（房价）
- **误差敏感性**：高价值房产的绝对误差影响更大
- **多维度特征**：面积、位置、房龄、装修等

### 2.2 数据集与模型准备

```python
from sklearn.datasets import fetch_california_housing
from sklearn.preprocessing import StandardScaler
from sklearn.linear_model import LinearRegression
from sklearn.ensemble import RandomForestRegressor

# 加载加州房价数据集（经典回归数据集）
housing = fetch_california_housing()
X_reg = housing.data
y_reg = housing.target
feature_names_reg = housing.feature_names

print("房价数据集信息：")
print(f"样本数: {X_reg.shape[0]}, 特征数: {X_reg.shape[1]}")
print(f"特征列表: {feature_names_reg}")
print(f"房价范围: ${y_reg.min():.0f} - ${y_reg.max():.0f}千美元")
print(f"平均房价: ${y_reg.mean():.0f}千美元")

# 数据标准化
scaler = StandardScaler()
X_reg_scaled = scaler.fit_transform(X_reg)

# 划分训练集/测试集
X_train_reg, X_test_reg, y_train_reg, y_test_reg = train_test_split(
    X_reg_scaled, y_reg, test_size=0.3, random_state=42
)

# 训练回归模型
linear_model = LinearRegression()
linear_model.fit(X_train_reg, y_train_reg)

rf_model = RandomForestRegressor(n_estimators=100, random_state=42)
rf_model.fit(X_train_reg, y_train_reg)

# 预测
y_pred_linear = linear_model.predict(X_test_reg)
y_pred_rf = rf_model.predict(X_test_reg)
```

### 2.3 回归评估指标计算

```python
from sklearn.metrics import mean_squared_error, mean_absolute_error, r2_score
import seaborn as sns

def calculate_regression_metrics(y_true, y_pred, model_name):
    """计算回归模型的各项评估指标"""
    mse = mean_squared_error(y_true, y_pred)
    rmse = np.sqrt(mse)
    mae = mean_absolute_error(y_true, y_pred)
    r2 = r2_score(y_true, y_pred)
    
    # 平均绝对百分比误差（MAPE）
    mape = np.mean(np.abs((y_true - y_pred) / np.maximum(np.abs(y_true), 1e-8))) * 100
    
    return {
        '模型': model_name,
        'MSE': mse,
        'RMSE': rmse,
        'MAE': mae,
        'R²': r2,
        'MAPE%': mape
    }

# 计算两个模型的指标
metrics_linear = calculate_regression_metrics(y_test_reg, y_pred_linear, '线性回归')
metrics_rf = calculate_regression_metrics(y_test_reg, y_pred_rf, '随机森林')

# 创建对比表格
metrics_df = pd.DataFrame([metrics_linear, metrics_rf])
print("\n回归模型性能对比：")
print(metrics_df.to_string(index=False))
```

## 总结：如何选择评估指标

### 分类问题指标选择指南
| 业务场景 | 推荐指标 | 理由 | 注意事项 |
|----------|----------|------|----------|
| 类别平衡 | 准确率、AUC | 全面评估分类能力 | 样本分布均匀时有效 |
| 类别不平衡 | F1分数、精确率-召回率曲线 | 关注少数类性能 | 避免准确率陷阱 |
| 代价敏感 | 代价矩阵、Fβ分数 | 根据错误代价调整权重 | 需要业务定义代价 |
| 多分类 | 宏平均F1、加权F1 | 考虑所有类别 | 注意类别权重 |

### 回归问题指标选择指南
| 业务需求 | 推荐指标 | 理由 | 注意事项 |
|----------|----------|------|----------|
| 一般预测 | RMSE、R² | 综合评估误差和解释力 | RMSE对异常值敏感 |
| 稳健评估 | MAE、MAPE | 降低异常值影响 | MAPE在真实值接近0时不稳定 |
| 相对误差 | MAPE、SMAPE | 关注误差比例 | 适合量纲不同的比较 |
| 分布分析 | 分位数误差 | 评估不同区间的预测质量 | 计算复杂度较高 |

### 核心要点
1. **没有万能指标**：根据业务目标选择，而不是追求单一数字
2. **可视化是关键**：指标数字 + 图表 = 完整评估
3. **业务可解释性**：技术指标必须能转化为业务语言
4. **持续监控**：模型上线后仍需定期评估性能变化
