### 关键代码片段
```python
# 梯度下降核心更新代码
def gradient_descent_update(self, X, y):
    m = X.shape[0]
    y_pred = self.predict(X)
    error = y_pred - y
    gradient = (1/m) * np.dot(X.T, error)
    self.weights -= self.learning_rate * gradient
    return self.weights
```

### 技术要点总结
1. **数据预处理：** 必须添加偏置项（一列1）以学习截距
2. **特征缩放：** 梯度下降对特征尺度敏感，标准化可加速收敛
3. **学习率选择：** 过大导致震荡，过小收敛慢，可用学习率衰减策略
4. **收敛判断：** 可监控损失变化率或梯度范数

### 常见问题与解决方案
1. **损失值NaN：** 减小学习率，添加梯度裁剪
2. **收敛过慢：** 增大学习率，检查特征尺度
3. **过拟合：** 增加训练数据，添加正则化，简化模型
