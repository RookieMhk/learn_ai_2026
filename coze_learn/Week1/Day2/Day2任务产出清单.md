## 常见问题与解决

### Q1: 运行代码时报错"ModuleNotFoundError: No module named 'numpy'"
**解决**：安装NumPy和Matplotlib
```bash
pip install numpy matplotlib
```

### Q2: 手动矩阵乘法运行时间太长（如200×200超过10秒）
**解决**：这是正常现象，体现了NumPy的优化优势。可以减小测试规模或仅运行到100×100。

### Q3: 幂迭代法结果与NumPy差异较大
**解决**：
1. 检查矩阵是否对称（对称矩阵特征值为实数）
2. 增加迭代次数（如100次）
3. 检查随机初始向量是否合理

### Q4: 可视化图表显示异常
**解决**：
1. 确保Matplotlib版本≥3.3
2. 检查中文字体设置
3. 尝试在Jupyter Notebook中运行
