## 一、核心概念精讲

### 1.1 向量与向量空间
**数学本质**：向量是具有大小和方向的量，向量空间是所有向量的集合及其上的运算规则。

**关键知识点**：
- **向量表示**：`v = [v₁, v₂, ..., vₙ] ∈ ℝⁿ`
- **基本运算**：加法、数乘、点积（内积）、范数（长度）
- **几何意义**：方向、长度、夹角余弦（cosθ = (a·b)/(‖a‖‖b‖)）

**AI应用场景**：
- **词嵌入（Word Embedding）**：词语语义表示为高维向量，语义相似度≈向量余弦相似度
  ```python
  # 示例：king - man + woman ≈ queen
  vec_king = [0.2, 0.8, -0.3]
  vec_man = [0.1, 0.7, -0.2]
  vec_woman = [0.3, 0.6, -0.4]
  # 计算结果应接近queen的向量
  ```
- **特征向量**：数据点的抽象表示，用于聚类、分类
- **梯度向量**：优化算法中的更新方向，指向损失函数下降最快方向

### 1.2 矩阵与矩阵运算
**数学本质**：矩阵是线性变换的数值表示，矩阵乘法对应线性变换的复合。

**关键知识点**：
- **矩阵表示**：`A ∈ ℝ^{m×n}`，m行n列的矩形数组
- **核心运算**：
  - **矩阵乘法**：`C = AB`，`Cᵢⱼ = Σₖ AᵢₖBₖⱼ`
  - **转置**：`Aᵀ`，行列互换
  - **逆矩阵**：`AA⁻¹ = I`（仅方阵且可逆时）
- **特殊矩阵**：单位矩阵、对角矩阵、对称矩阵、正交矩阵

**AI应用场景**：
- **神经网络权重矩阵**：连接层与层的线性变换参数
  ```python
  # 单层前向传播：z = Wx + b
  W = np.random.randn(256, 128)  # 权重矩阵
  x = np.random.randn(128, 1)    # 输入向量
  b = np.random.randn(256, 1)    # 偏置向量
  z = W @ x + b
  ```
- **图像数据表示**：RGB图像为三维张量（高度×宽度×通道）
- **数据集组织**：样本×特征矩阵，每行一个样本，每列一个特征

### 1.3 特征值与特征向量
**数学定义**：对于方阵A，若存在非零向量v和标量λ，使`Av = λv`，则λ为特征值，v为对应特征向量。

**几何解释**：特征向量是线性变换中方向不变的向量，特征值是沿该方向的缩放因子。

**计算示例**：
```python
import numpy as np

A = np.array([[4, 2], [1, 3]])
eigenvalues, eigenvectors = np.linalg.eig(A)
# eigenvalues ≈ [5, 2]
# eigenvectors ≈ [[0.894, -0.707], [0.447, 0.707]]
```

**AI应用场景**：
- **主成分分析（PCA）**：数据降维，提取最大方差方向（即最大特征值对应的特征向量）
- **PageRank算法**：网页重要性排名，特征向量对应稳态分布
- **协方差矩阵分析**：多元统计中揭示变量间线性关系

### 1.4 奇异值分解（SVD）
**数学公式**：`A = UΣVᵀ`，其中U、V为正交矩阵，Σ为对角矩阵（奇异值）。

**分解意义**：
- **U**：左奇异向量，代表输入空间的正交基
- **Σ**：奇异值，表示每个方向的重要性（按降序排列）
- **Vᵀ**：右奇异向量，代表输出空间的正交基

**计算示例**：
```python
A = np.array([[1, 2], [3, 4], [5, 6]])
U, Sigma, Vt = np.linalg.svd(A, full_matrices=False)
# A ≈ U @ np.diag(Sigma) @ Vt
```

**AI应用场景**：
- **推荐系统**：矩阵补全，协同过滤（Netflix Prize经典方法）
- **图像压缩**：保留前k个奇异值，近似原图像，大幅减少存储
- **自然语言处理**：潜在语义分析（LSA），文档-词语矩阵分解

## 二、LLM关联思考引导

### 2.1 Transformer注意力机制中的线性代数
**核心问题**：自注意力计算公式`Attention(Q, K, V) = softmax(QKᵀ/√dₖ)V`涉及哪些线性代数运算？

**思考步骤**：
1. **矩阵乘法**：`QKᵀ`是两个矩阵的乘积，计算查询与键的相似度
   - Q形状：`[n_seq, d_k]`，Kᵀ形状：`[d_k, n_seq]`
   - 结果：`[n_seq, n_seq]`的注意力分数矩阵
   
2. **缩放运算**：除以`√dₖ`是标量除法，防止点积过大导致softmax梯度消失

3. **矩阵乘法**：注意力权重与V相乘，`[n_seq, n_seq] × [n_seq, d_v] → [n_seq, d_v]`

**编码实现**：
```python
def attention(Q, K, V, mask=None):
    d_k = Q.shape[-1]
    scores = Q @ K.transpose(-2, -1) / np.sqrt(d_k)  # 矩阵乘法+缩放
    if mask is not None:
        scores = scores + mask
    attention_weights = softmax(scores)              # 非线性激活
    output = attention_weights @ V                   # 矩阵乘法
    return output, attention_weights
```

### 2.2 特征值分解与模型稳定性
**思考问题**：为什么在训练深度神经网络时，权重矩阵的特征值分布会影响训练稳定性？

**分析角度**：
- **梯度爆炸/消失**：反复应用线性变换`W`时，若最大特征值`|λ_max| > 1`，激活值指数增长（爆炸）；若`|λ_max| < 1`，指数衰减（消失）
- **初始化策略**：Xavier/He初始化旨在使权重矩阵的特征值接近1，保持信号传播稳定
- **归一化技术**：BatchNorm/LayerNorm通过标准化激活值，间接控制特征值影响

### 2.3 SVD在模型压缩中的应用
**实践思考**：如何利用SVD压缩全连接层的参数数量？

**方案设计**：
1. **全连接层**：`y = Wx`，`W ∈ ℝ^{m×n}`
2. **SVD分解**：`W = UΣVᵀ`
3. **低秩近似**：保留前k个奇异值，`W̃ = U[:,:k] @ Σ[:k,:k] @ Vᵀ[:k,:]`
4. **参数减少**：原参数`m×n` → 近似参数`(m×k + k + n×k)`
5. **精度权衡**：通过奇异值衰减曲线选择k，平衡压缩比与精度损失
