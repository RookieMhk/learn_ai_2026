# 泰坦尼克号数据集获取与加载指南

## 数据集简介

泰坦尼克号数据集是数据科学入门最经典的案例数据集，包含了1912年泰坦尼克号沉船事件中乘客的个人信息与生存状况。该数据集常用于数据清洗、特征工程、机器学习分类等教学场景。

数据集包含以下特征：

| 字段名 | 数据类型 | 描述 |
|--------|----------|------|
| PassengerId | 整型 | 乘客ID |
| Survived | 整型 | 是否幸存（0=否，1=是） |
| Pclass | 整型 | 船舱等级（1=头等舱，2=二等舱，3=三等舱） |
| Name | 字符串 | 乘客姓名 |
| Sex | 字符串 | 性别（male/female） |
| Age | 浮点型 | 年龄 |
| SibSp | 整型 | 同船兄弟姐妹/配偶数量 |
| Parch | 整型 | 同船父母/子女数量 |
| Ticket | 字符串 | 船票编号 |
| Fare | 浮点型 | 票价 |
| Cabin | 字符串 | 船舱号 |
| Embarked | 字符串 | 登船港口（C=Cherbourg, Q=Queenstown, S=Southampton） |

## 获取方式（三种途径）

### 方式一：使用Seaborn内置数据集（推荐）

Seaborn库内置了泰坦尼克号数据集，无需额外下载，数据稳定可靠。

```python
import seaborn as sns
import pandas as pd

# 加载数据集
titanic = sns.load_dataset('titanic')
print(f"数据集形状: {titanic.shape}")
print(titanic.head())
```

**安装Seaborn（如未安装）**：
```bash
pip install seaborn
```

### 方式二：从Kaggle下载

如果需要进行完整的数据分析练习，建议从Kaggle下载完整数据集（包含训练集和测试集）。

1. **访问Kaggle页面**：https://www.kaggle.com/c/titanic/data
2. **下载文件**：点击"Download All"下载压缩包
3. **解压后加载**：

```python
import pandas as pd

# 加载训练集
train_df = pd.read_csv('train.csv')
# 加载测试集
test_df = pd.read_csv('test.csv')

print(f"训练集形状: {train_df.shape}")
print(f"测试集形状: {test_df.shape}")
```

### 方式三：直接URL下载

从网络直接下载CSV文件（适合无法访问Kaggle的用户）：

```python
import pandas as pd
import io
import requests

# 泰坦尼克号数据集URL（GitHub托管）
url = 'https://raw.githubusercontent.com/datasciencedojo/datasets/master/titanic.csv'

try:
    titanic = pd.read_csv(url)
    print(f"成功加载数据集，形状: {titanic.shape}")

    # 备用URL
    backup_url = 'https://web.stanford.edu/class/archive/cs/cs109/cs109.1166/stuff/titanic.csv'
    titanic = pd.read_csv(backup_url)
    print(f"从备用URL加载成功，形状: {titanic.shape}")
```

### 内存优化（大数据集时）
对于大型数据集，可以使用以下技巧减少内存占用：

```python
def reduce_memory_usage(df):
    """降低DataFrame内存占用"""
    start_mem = df.memory_usage().sum() / 1024**2
    print(f"初始内存占用: {start_mem:.2f} MB")
    
    for col in df.columns:
        col_type = df[col].dtype
        
        if col_type != object:
            c_min = df[col].min()
            c_max = df[col].max()
            
            if str(col_type)[:3] == 'int':
                if c_min > np.iinfo(np.int8).min and c_max < np.iinfo(np.int8).max:
                    df[col] = df[col].astype(np.int8)
                elif c_min > np.iinfo(np.int16).min and c_max < np.iinfo(np.int16).max:
                    df[col] = df[col].astype(np.int16)
                elif c_min > np.iinfo(np.int32).min and c_max < np.iinfo(np.int32).max:
                    df[col] = df[col].astype(np.int32)
                else:
                    df[col] = df[col].astype(np.int64)
            else:
                if c_min > np.finfo(np.float16).min and c_max < np.finfo(np.float16).max:
                    df[col] = df[col].astype(np.float16)
                elif c_min > np.finfo(np.float32).min and c_max < np.finfo(np.float32).max:
                    df[col] = df[col].astype(np.float32)
                else:
                    df[col] = df[col].astype(np.float64)
        else:
            df[col] = df[col].astype('category')
    
    end_mem = df.memory_usage().sum() / 1024**2
    print(f"优化后内存占用: {end_mem:.2f} MB")
    print(f"内存减少: {100 * (start_mem - end_mem) / start_mem:.1f}%")
    
    return df

# 使用示例
titanic = reduce_memory_usage(titanic)
```

## 常见问题解决

1. **加载失败**：检查网络连接，尝试使用方式一（Seaborn内置数据）
2. **内存不足**：使用上述内存优化函数，或只加载部分数据
3. **编码问题**：如果遇到编码错误，尝试指定编码格式：`pd.read_csv('file.csv', encoding='latin1')`
