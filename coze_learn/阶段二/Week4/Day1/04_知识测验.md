## 一、基础概念题（5题）

### Q1. RNN的核心创新是什么？

```
A. 使用卷积操作处理序列
B. 隐藏状态在时间步之间传递，赋予网络"记忆"
C. 使用注意力机制捕捉长距离依赖
D. 将序列并行化处理
```

<details>
<summary>点击查看答案</summary>

**答案：B**

**解析**：RNN的核心创新是隐藏状态的时间传播机制。不同于传统前馈网络，RNN在每个时间步接收当前输入和上一时间步的隐藏状态，产生新的隐藏状态，从而实现信息的跨时间传递。

</details>

---

### Q2. 什么是"权重共享"？为什么RNN要使用权重共享？

```
A. 多个层使用相同的权重，减少参数
B. 在序列的不同位置使用相同的权重矩阵，实现泛化
C. 将权重在训练中共享，加快速度
D. 权重在CPU和GPU之间共享
```

<details>
<summary>点击查看答案</summary>

**答案：B**

**解析**：权重共享意味着无论序列有多长，模型使用相同的权重矩阵处理每个时间步。这使得RNN可以处理任意长度的序列，并且具有很好的泛化能力——训练学到的模式可以应用于任意长度的输入。

</details>

---

### Q3. 标准RNN面临的主要问题是什么？

```
A. 过拟合
B. 梯度消失/爆炸
C. 内存不足
D. 训练速度太慢
```

<details>
<summary>点击查看答案</summary>

**答案：B**

**解析**：标准RNN面临严重的梯度消失和梯度爆炸问题。这是因为在BPTT中，梯度需要通过多个时间步的链式乘法传播。当权重特征值小于1时梯度消失，大于1时梯度爆炸，导致网络难以学习长期依赖。

</details>

---

### Q4. LSTM解决梯度消失的核心机制是什么？

```
A. 使用ReLU激活函数
B. 添加残差连接
C. 细胞状态的加性更新结构
D. 使用更大的隐藏层
```

<details>
<summary>点击查看答案</summary>

**答案：C**

**解析**：LSTM通过细胞状态(Cell State)的加性更新解决梯度消失。细胞状态的更新公式是 `C_t = f_t ⊙ C_{t-1} + i_t ⊙ C̃_t`，这是加法而非乘法。梯度可以沿着细胞状态直接传播，当遗忘门接近1时几乎没有损耗。

</details>

---

### Q5. GRU相比LSTM的主要简化是什么？

```
A. 减少隐藏层维度
B. 合并遗忘门和输入门为更新门，去除独立的细胞状态
C. 减少门控数量
D. B和C都是
```

<details>
<summary>点击查看答案</summary>

**答案：D**

**解析**：GRU做了两处关键简化：1) 将遗忘门和输入门合并为单一的更新门；2) 移除了独立的细胞状态，直接在隐藏状态中维护记忆。这使得GRU参数量减少约25%，计算效率更高。

</details>

## 二、数学原理题（5题）

### Q6. LSTM中遗忘门的作用是什么？（用数学公式表示）

```
A. σ(W_f · [h_{t-1}, x_t])：决定保留多少之前的记忆
B. tanh(W_C · [h_{t-1}, x_t])：生成新的候选记忆
C. σ(W_o · [h_{t-1}, x_t])：决定输出多少
D. h_t = (1-z_t) ⊙ h_{t-1} + z_t ⊙ h̃_t：更新隐藏状态
```

<details>
<summary>点击查看答案</summary>

**答案：A**

**解析**：遗忘门 f_t = σ(W_f · [h_{t-1}, x_t]) 决定从细胞状态中遗忘多少信息。值接近1时保留旧信息，接近0时遗忘旧信息。

</details>

---

### Q7. RNN隐藏状态更新的数学表达式是？

```
A. h_t = σ(W · x_t + b)
B. h_t = tanh(W_hh · h_{t-1} + W_xh · x_t + b_h)
C. h_t = softmax(W · h_{t-1})
D. h_t = max(0, W · x_t + b)
```

<details>
<summary>点击查看答案</summary>

**答案：B**

**解析**：标准RNN的隐藏状态更新公式为：
h_t = tanh(W_hh · h_{t-1} + W_xh · x_t + b_h)
其中tanh将结果压缩到[-1,1]区间，引入非线性。

</details>

---

### Q8. GRU中更新门 z_t 的取值范围和作用是？

```
A. [0,∞)，控制模型学习速度
B. (-1,1)，决定保留多少旧记忆
C. [0,1]，控制新旧隐藏状态的混合比例
D. 任意值，注意力权重
```

<details>
<summary>点击查看答案</summary>

**答案：C**

**解析**：更新门 z_t = σ(W_z · [h_{t-1}, x_t])，由于sigmoid函数，输出范围是[0,1]。最终隐藏状态为 h_t = (1-z_t) ⊙ h_{t-1} + z_t ⊙ h̃_t，所以z_t接近1时更多采用新记忆，接近0时更多保留旧记忆。

</details>

---

### Q9. LSTM细胞状态更新的公式是什么？

```
A. C_t = W · C_{t-1} + x_t
B. C_t = f_t ⊙ C_{t-1} + i_t ⊙ C̃_t
C. C_t = tanh(C_{t-1}) + tanh(C̃_t)
D. C_t = softmax(C_{t-1})
```

<details>
<summary>点击查看答案</summary>

**答案：B**

**解析**：这是LSTM的核心公式。细胞状态通过遗忘门(f_t)保留部分旧状态，通过输入门(i_t)添加新的候选记忆(C̃_t)。加性结构使得梯度可以有效传播。

</details>

---

### Q10. 为什么LSTM的输出门 o_t 是必要的？

```
A. 为了控制梯度流动
B. 为了决定从细胞状态中提取什么信息作为输出
C. 为了减少参数量
D. 输出门不是必要的，可以省略
```

<details>
<summary>点击查看答案</summary>

**答案：B**

**解析**：输出门 o_t = σ(W_o · [h_{t-1}, x_t]) 决定从细胞状态中输出多少信息。最终隐藏状态 h_t = o_t ⊙ tanh(C_t) 将细胞状态通过tanh压缩后，由输出门控制输出量。

</details>

## 三、架构与应用题（5题）

### Q11. Seq2Seq模型中，"编码器-解码器"架构的作用是什么？

```
A. 编码器压缩输入，解码器恢复输出
B. 编码器提取特征，解码器生成序列
C. 编码器并行处理，解码器顺序处理
D. 编码器和解码器功能相同
```

<details>
<summary>点击查看答案</summary>

**答案：B**

**解析**：Seq2Seq的编码器将变长输入序列编码为固定维度的上下文向量，解码器基于这个向量逐步生成变长的输出序列。这是机器翻译、文本摘要等任务的基础架构。

</details>

---

### Q12. 注意力机制解决了Seq2Seq的什么问题？

```
A. 训练速度慢
B. 内存不足
C. 信息瓶颈（所有信息压缩到单一上下文向量）
D. 梯度消失
```

<details>
<summary>点击查看答案</summary>

**答案：C**

**解析**：标准Seq2Seq将整个输入序列压缩到单一上下文向量，造成信息瓶颈。注意力机制让解码器在生成每个输出时，可以"关注"输入序列的不同位置，有效解决了这一问题。

</details>

### Q13. 以下哪种场景最适合使用GRU而不是LSTM？

```
A. 需要精细控制记忆的复杂任务
B. 实时性要求高的在线预测场景
C. 需要记忆超过1000步的长期依赖
D. 需要最小化参数量的嵌入式部署
```

<details>
<summary>点击查看答案</summary>

**答案：B、D**

**解析**：GRU的优势在于：1) 参数量少25%，适合资源受限场景；2) 计算效率高，适合实时应用。但对于需要精细记忆控制或极长依赖的任务，LSTM可能更合适。

</details>

### Q14. 什么是"双向LSTM"？它适用于什么场景？

```
A. 两个方向相反的LSTM叠加，用于需要知道未来信息的场景
B. 前后两个方向的LSTM，可以同时利用过去和未来的上下文
C. 将LSTM翻转后使用，用于数据增强
D. 两层LSTM堆叠，增加模型容量
```

<details>
<summary>点击查看答案</summary>

**答案：B**

**解析**：双向LSTM由前向和后向两个LSTM组成。前向LSTM从左到右处理序列，后向LSTM从右到左处理。对于文本分类等任务，双向可以同时利用过去和未来的上下文信息。

</details>

### Q15. 2025年Bengio团队提出的minLSTM/minGRU的核心改进是什么？

```
A. 增加了更多门控
B. 去除了门控对隐藏状态的依赖，支持并行训练
C. 扩大了隐藏层维度
D. 引入了注意力机制
```

<details>
<summary>点击查看答案</summary>

**答案：B**

**解析**：minLSTM/minGRU的关键创新是去除门控对先前隐藏状态的依赖，使门控仅依赖当前输入。这允许使用并行扫描算法进行训练，大大提高训练效率，同时保持与原始模型相当的性能。

</details>
