## 🧠 知识回顾

### 反向传播流程

```
输入数据 → 前向传播 → 计算损失 → 反向传播 → 更新参数
            ↓                        ↓
        得到输出              从后向前计算梯度
```

### 梯度更新公式（记忆）

**链式法则**：
$$(∂ L)/(∂ w) = {∂ L}{∂ {y}} · {∂ {y}}{∂ z} · (∂ z)/(∂ w)$$

**Adam 更新**：
$$θ_ₜ₊₁ = θₜ - η/(√vₜ + ε) · mₜ$$
