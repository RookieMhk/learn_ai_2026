# 优化器原理对比

> **学习目标**：理解不同优化算法的核心思想、优缺点及适用场景

---

## 1. 优化器概述

### 1.1 问题背景

神经网络的参数更新目标是最小化损失函数：
$$min_θ {L}(θ)$$

梯度下降法通过沿负梯度方向更新参数：
$$θ_ₜ₊₁ = θₜ - η ∇_θ {L}(θₜ)$$

其中 $η$ 为学习率。

### 1.2 优化器分类

```
优化器
├── 一阶方法（只使用一阶导数）
│   ├── SGD
│   ├── SGD + Momentum
│   ├── Nesterov Accelerated Gradient (NAG)
│   ├── Adagrad
│   ├── RMSprop
│   └── Adam
└── 二阶方法（使用二阶导数）
    └── Newton, L-BFGS（计算量大，较少使用）
```

---

## 2. 基础优化器

### 2.1 SGD（随机梯度下降）

**更新公式**：
$$θ_ₜ₊₁ = θₜ - η · gₜ$$

其中 $gₜ = ∇_θ {L}(θₜ)$

**特点**：
- 优点：简单高效，内存占用小
- 缺点：收敛慢，容易陷入局部最优，学习率需手动调整

**代码实现**：
```python
theta = theta - lr * gradient
```

### 2.2 Mini-batch SGD

**改进**：使用小批量数据进行梯度计算
$$θ_ₜ₊₁ = θₜ - η · 1/m Σ_ᵢ₌₁ᵐ ∇_θ {L}(x⁽ⁱ⁾, y⁽ⁱ⁾; θₜ)$$

**优点**：
- 梯度估计更稳定
- 充分利用GPU并行计算能力
- 收敛曲线更平滑

---

## 3. 带动量的优化器

### 3.1 SGD with Momentum

**物理直觉**：将参数更新类比为小球在曲面滚动，累积惯性。

**更新公式**：
$$vₜ = β v_ₜ₋₁ + (1 - β) · gₜ$$
$$θ_ₜ₊₁ = θₜ - η · vₜ$$

其中：
- $vₜ$：动量项（速度）
- $β$：动量系数，通常取 0.9

**直观理解**：
```
不带动量：    带动量（β=0.9）：
    ↓           
    ↓           
    ↓           
    ↓→→→        
    
    ↓↓↓↓↓↓      ↓↓↓
              →→→→→
```

**代码实现**：
```python
v = beta * v + (1 - beta) * gradient
theta = theta - lr * v
```

### 3.2 Nesterov Accelerated Gradient (NAG)

**改进**：先看未来的位置，再决定更新方向。

**更新公式**：
$$vₜ = β v_ₜ₋₁ + (1 - β) · ∇_θ {L}(θₜ - η β v_ₜ₋₁)$$
$$θ_ₜ₊₁ = θₜ - η · vₜ$$

**对比**：
| 方法 | 梯度计算位置 | 预见性 |
|------|-------------|--------|
| SGD+Momentum | 当前 $θₜ$ | 无 |
| NAG | $θₜ - ηβ v_ₜ₋₁$ | 有 |

---

## 4. 自适应学习率优化器

### 4.1 Adagrad

**核心思想**：对稀疏参数使用大更新，频繁参数使用小更新。

**更新公式**：
$$Gₜ = G_ₜ₋₁ + gₜ²$$
$$θ_ₜ₊₁ = θₜ - η/(√Gₜ + ε) gₜ$$

其中：
- $Gₜ$：累计梯度平方和
- $ε$：防止除零，通常取 $10⁻⁸$

**优点**：适合稀疏数据
**缺点**：学习率持续衰减，后期可能过小

### 4.2 RMSprop

**改进**：使用指数加权移动平均替代累加。

**更新公式**：
$$E[g²]ₜ = β E[g²]_ₜ₋₁ + (1 - β) · gₜ²$$
$$θ_ₜ₊₁ = θₜ - η/(√E[g²]ₜ + ε) gₜ$$

**参数选择**：
- $β$：通常取 0.9
- $η$：通常取 0.001

**代码实现**：
```python
cache = beta * cache + (1 - beta) * gradient**2
theta = theta - lr * gradient / (np.sqrt(cache) + eps)
```

### 4.3 Adadelta

**改进**：不使用全局学习率，而是使用参数更新的二阶矩。

**更新公式**：
$$E[Δθ²]ₜ = β E[Δθ²]_ₜ₋₁ + (1-β) Δθₜ²$$
$$θ_ₜ₊₁ = θₜ - (√E[Δθ²]_ₜ₋₁ + ε)/(√E[g²]ₜ + ε) gₜ$$

**优点**：无需手动设置学习率

## 5. Adam：综合最优

### 5.1 算法原理

Adam = Adaptive + Momentum，同时利用：
- **动量项**：一阶矩估计（均值）
- **自适应学习率**：二阶矩估计（方差）

**更新公式**：

**Step 1**: 计算梯度
$$gₜ = ∇_θ {L}(θₜ)$$

**Step 2**: 更新一阶矩估计（动量）
$$mₜ = β₁ m_ₜ₋₁ + (1 - β₁) gₜ$$

**Step 3**: 更新二阶矩估计（自适应学习率）
$$vₜ = β₂ v_ₜ₋₁ + (1 - β₂) gₜ²$$

**Step 4**: 偏差修正
$${m}ₜ = mₜ/(1 - β₁ᵗ)$$
$${v}ₜ = vₜ/(1 - β₂ᵗ)$$

**Step 5**: 参数更新
$$θ_ₜ₊₁ = θₜ - {η}{√{vₜ} + ε} {m}ₜ$$

### 5.2 参数默认值

| 参数 | 默认值 | 说明 |
|------|--------|------|
| $η$ | 0.001 | 学习率 |
| $β₁$ | 0.9 | 一阶矩衰减 |
| $β₂$ | 0.999 | 二阶矩衰减 |
| $ε$ | $10⁻⁸$ | 数值稳定项 |

### 5.3 偏差修正的必要性

初始时 $m₀ = 0, v₀ = 0$，需要修正：

$${m}₁ = (1-β₁)/(1-β₁¹) m₁ ≈ m₁$$

当 $t$ 增大时，$β₁ᵗ → 0$，修正效果减弱。

## 6. 算法对比

### 6.1 核心公式对比

| 优化器 | 更新方向 | 学习率 |
|--------|---------|--------|
| SGD | $-gₜ$ | 全局 |
| SGD+Momentum | $-1/(1-β)[vₜ]$ | 全局 |
| Adagrad | $-gₜ / √Gₜ$ | 自适应 |
| RMSprop | $-gₜ / √E[gₜ²]$ | 自适应 |
| Adam | $-mₜ / √vₜ$ | 自适应 |

### 6.2 优缺点对比

| 优化器 | 优点 | 缺点 | 适用场景 |
|--------|------|------|----------|
| SGD | 泛化好，可微调 | 收敛慢，需调参 | 图像分类 |
| SGD+Momentum | 加速收敛，减少震荡 | 需调参 | 通用 |
| NAG | 更好的预见性 | 实现复杂 | 优化问题 |
| Adagrad | 适合稀疏数据 | 学习率衰减 | NLP、推荐 |
| RMSprop | 解决Adagrad衰减问题 | 效果不稳定 | RNN |
| Adam | 综合最优，默认好用 | 可能泛化差 | 深度学习首选 |

### 6.3 收敛轨迹对比

```
        最优点
           ★
          / \
    SGD  /   \  Adam
        /     \
       /   ←→  \
      /  震荡   \  快速收敛
     /           \
    ●─────────────●
    
RMSprop: 中等速度
Adagrad: 越来越慢
```

### 6.4 学习率调度建议

| 场景 | 优化器 | 学习率 |
|------|--------|--------|
| 默认 | Adam | 0.001 |
| 图像分类 | SGD+Momentum | 0.01 |
| Transformer | AdamW | 0.0001-0.001 |
| LSTM | RMSprop/Adam | 0.001 |

## 7. 实践技巧

### 7.1 学习率选择

```
初始学习率搜索范围：
- SGD: [0.01, 0.1, 1.0]
- Adam: [0.0001, 0.001, 0.01]
- 大batch: 可增大学习率 (linear scaling)
```

### 7.2 Warmup策略

**问题**：训练初期参数随机，梯度可能很大

**解决方案**：逐渐增大学习率
```python
if step < warmup_steps:
    lr = base_lr * step / warmup_steps
else:
    lr = base_lr * decay_factor
```

### 7.3 权重衰减（Weight Decay）

L2正则化的另一种形式：
```python
theta = theta * (1 - lr * lambda) - lr * gradient
```

AdamW = Adam + 正确的权重衰减：
```python
theta = theta - lr * (m_t / (sqrt(v_t) + eps) + lambda * theta)
```

## 8. 优化器选择指南

```
开始新项目？
    │
    ├─ 是通用深度学习？→ Adam (默认选择)
    │
    ├─ 是图像分类/CNN？→ SGD + Momentum + Warmup
    │
    ├─ 是NLP/Transformer？→ AdamW + Weight Decay
    │
    ├─ 是RNN/LSTM？→ RMSprop 或 Adam
    │
    └─ 资源有限？→ SGD + Momentum（最终收敛更好）
```

---

## 9. 总结

| 优化器 | 记忆口诀 |
|--------|----------|
| SGD | 简单直接，慢但稳 |
| Momentum | 借助惯性，加速下山 |
| Adagrad | 记录历史，稀疏友好 |
| RMSprop | 遗忘过去，自适应步 |
| Adam | 动量+自适应，全能选手 |

> 💡 **实战建议**：先用 Adam 快速验证想法，再换 SGD+Momentum 调优最终性能
