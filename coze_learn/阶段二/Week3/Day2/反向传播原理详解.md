# 反向传播原理详解

> **学习目标**：深入理解反向传播算法的数学本质，掌握链式法则在神经网络中的应用

---

## 1. 链式法则：反向传播的数学基础

### 1.1 导数回顾

对于函数 $y = f(x)$，导数定义为：
$$\frac{dy}{dx} = \lim_{\Delta x \to 0} \frac{f(x + \Delta x) - f(x)}{\Delta x}$$

导数表示：当输入变化时，输出如何变化。

### 1.2 一维链式法则

对于复合函数 $y = f(g(x))$：
$$\frac{dy}{dx} = \frac{dy}{dg} \cdot \frac{dg}{dx}$$

### 1.3 多元链式法则（核心）

对于 $z = f(u_1, u_2, ...)$，其中 $u_i = g_i(x)$：
$$\frac{\partial z}{\partial x} = \sum_i \frac{\partial z}{\partial u_i} \cdot \frac{\partial u_i}{\partial x}$$

**示例推导**：

给定：
- $z = u_1^2 + u_2 \cdot u_1$
- $u_1 = 2x + 1$
- $u_2 = x^2$

求解 $\frac{\partial z}{\partial x}$：

$$\frac{\partial z}{\partial u_1} = 2u_1 + u_2$$
$$\frac{\partial z}{\partial u_2} = u_1$$
$$\frac{\partial u_1}{\partial x} = 2$$
$$\frac{\partial u_2}{\partial x} = 2x$$

因此：
$$\frac{\partial z}{\partial x} = (2u_1 + u_2) \cdot 2 + u_1 \cdot 2x$$

## 2. 计算图：可视化神经网络

### 2.1 什么是计算图

计算图是神经网络的数学表示，每个节点代表一个操作，每条边代表数据流动。

```
输入 x ──→ [×w₁] ──→ [+] ──→ [σ] ──→ ŷ
            ↑          ↑
         权重 w₁    偏置 b
```

### 2.2 前向传播 vs 反向传播

| 阶段 | 方向 | 计算内容 |
|------|------|----------|
| 前向传播 | 左→右 | 计算输出 $\hat{y}$ |
| 反向传播 | 右→左 | 计算梯度 $\frac{\partial L}{\partial w}$ |

### 2.3 典型MLP计算图

```
输入层        隐藏层        输出层
x₁ ──────┐
         ├──→ [Σ] ──→ [σ] ──┐
x₂ ──────┤   (z₁)          ├──→ [Σ] ──→ [σ] ──→ ŷ
         ├──→ [Σ] ──→ [σ] ──┘   (z₃)
x₃ ──────┘   (z₂)
```

## 3. 反向传播算法详解

### 3.1 单层神经网络的梯度计算

**前向传播**：
$$z = \mathbf{w} \cdot \mathbf{x} + b$$
$$\hat{y} = \sigma(z)$$

**损失函数**（MSE）：
$$L = \frac{1}{2}(y - \hat{y})^2$$

**反向传播推导**：

**Step 1**：输出层梯度
$$\frac{\partial L}{\partial \hat{y}} = \hat{y} - y$$

**Step 2**：激活函数梯度
$$\frac{\partial \hat{y}}{\partial z} = \sigma'(z) = \sigma(z)(1 - \sigma(z))$$

**Step 3**：加权求和梯度
$$\frac{\partial z}{\partial w_i} = x_i, \quad \frac{\partial z}{\partial b} = 1$$

**Step 4**：综合梯度（链式法则）
$$\frac{\partial L}{\partial w_i} = \frac{\partial L}{\partial \hat{y}} \cdot \frac{\partial \hat{y}}{\partial z} \cdot \frac{\partial z}{\partial w_i}$$

展开：
$$\frac{\partial L}{\partial w_i} = (\hat{y} - y) \cdot \sigma'(z) \cdot x_i$$

### 3.2 多层神经网络的梯度计算

对于 $L$ 层网络：

```
输入 x → [Layer₁] → [Layer₂] → ... → [Layerₗ] → ... → [Layerᴸ] → 输出
              ↑            ↑                    ↑
            梯度δ¹       梯度δ²                梯度δᴸ
```

**通用递推公式**：

第 $l$ 层的梯度 $\delta^l$：
$$\delta^l = \frac{\partial L}{\partial z^l}$$

**从后向前传播**：
$$\delta^{l} = (W^{l+1})^T \delta^{l+1} \odot \sigma'(z^l)$$

其中 $\odot$ 表示逐元素乘法。

## 4. 梯度消失与梯度爆炸

### 4.1 梯度消失（Vanishing Gradient）

**问题描述**：
$$\frac{\partial L}{\partial w^{(1)}} = \frac{\partial L}{\partial \hat{y}} \cdot \prod_{i=2}^{L} \sigma'(z^i) \cdot \prod_{i=1}^{L-1} W^i$$

当 $|W\sigma'(z)| < 1$ 时，梯度逐层指数衰减。

**影响**：
- 靠近输入层的权重几乎不更新
- 网络难以学习深层特征
- Sigmoid/tanh 激活函数尤其容易出现

### 4.2 梯度爆炸（Exploding Gradient）

**问题描述**：
$$\frac{\partial L}{\partial w^{(1)}} = \frac{\partial L}{\partial \hat{y}} \cdot \prod_{i=2}^{L} \sigma'(z^i) \cdot \prod_{i=1}^{L-1} W^i$$

当 $|W\sigma'(z)| > 1$ 时，梯度指数增长。

**影响**：
- 参数更新过大
- 损失函数震荡/发散
- 数值溢出（NaN）

### 4.3 对比表格

| 现象 | 梯度消失 | 梯度爆炸 |
|------|----------|----------|
| 梯度值 | → 0 | → ∞ |
| 训练表现 | 前层不学习 | 震荡/发散 |
| 常见场景 | 深层网络+Sigmoid | 深层网络+大权重 |
| 解决方案 | ReLU/残差连接 | 梯度裁剪/LSTM |

## 5. 激活函数对梯度的影响

### 5.1 Sigmoid

$$\sigma(x) = \frac{1}{1 + e^{-x}}$$

$$\sigma'(x) = \sigma(x)(1 - \sigma(x))$$

**特点**：
- 输出范围：(0, 1)
- 最大导数：0.25（当 x=0 时）
- 深层网络梯度快速衰减

**梯度曲线**：
```
导数值
  0.25 ┤╮
       │ ╰─→ 最大值
  0.00 ┤──────────╯
       └──────────────→ x
        -10  0  10
```

### 5.2 Tanh

$$\tanh(x) = \frac{e^x - e^{-x}}{e^x + e^{-x}}$$

$$\tanh'(x) = 1 - \tanh^2(x)$$

**特点**：
- 输出范围：(-1, 1)
- 最大导数：1
- 零中心化，但仍有梯度消失

### 5.3 ReLU

$$f(x) = \max(0, x)$$

$$f'(x) = \begin{cases} 1 & x > 0 \\ 0 & x \leq 0 \end{cases}$$

**特点**：
- 缓解梯度消失（正区间梯度恒为1）
- 计算高效
- **dying ReLU**：负区间神经元可能永久死亡

### 5.4 梯度特性对比

| 激活函数 | 梯度范围 | 零中心 | 梯度消失风险 |
|----------|----------|--------|--------------|
| Sigmoid | (0, 0.25] | 否 | 高 |
| Tanh | (0, 1] | 是 | 中 |
| ReLU | {0, 1} | 否 | 低 |
| Leaky ReLU | (α, 1] | 否 | 低 |
| ELU | (α, 1] | 是 | 低 |

## 6. 实战：手推反向传播

### 6.1 示例网络

```
输入: x₁=1, x₂=2
权重: w₁=0.5, w₂=0.3, w₃=0.8, w₄=0.6
偏置: b₁=0.1, b₂=0.2
激活函数: Sigmoid
损失: MSE, 真实值 y=1
```

### 6.2 前向传播

**第一层**：
$$z_1 = w_1 x_1 + w_2 x_2 + b_1 = 0.5 \times 1 + 0.3 \times 2 + 0.1 = 1.2$$
$$a_1 = \sigma(z_1) = \frac{1}{1+e^{-1.2}} \approx 0.7685$$

$$z_2 = w_3 x_1 + w_4 x_2 + b_2 = 0.8 \times 1 + 0.6 \times 2 + 0.2 = 2.2$$
$$a_2 = \sigma(z_2) = \frac{1}{1+e^{-2.2}} \approx 0.9002$$

**第二层**（输出层，假设单输出）：
$$z_3 = w_5 a_1 + w_6 a_2 + b_3 = 0.4 \times 0.7685 + 0.7 \times 0.9002 + 0.1 = 1.047$$
$$\hat{y} = \sigma(z_3) = \frac{1}{1+e^{-1.047}} \approx 0.7400$$

**损失**：
$$L = \frac{1}{2}(y - \hat{y})^2 = \frac{1}{2}(1 - 0.74)^2 = 0.0338$$

### 6.3 反向传播

**Step 1**：输出层
$$\delta_3 = \frac{\partial L}{\partial z_3} = (0.74 - 1) \times 0.74 \times 0.26 = -0.0498$$

**Step 2**：第二层权重梯度
$$\frac{\partial L}{\partial w_5} = \delta_3 \times a_1 = -0.0498 \times 0.7685 = -0.0383$$
$$\frac{\partial L}{\partial w_6} = \delta_3 \times a_2 = -0.0498 \times 0.9002 = -0.0448$$

**Step 3**：传播到第一层
$$\delta_1 = \delta_3 \times w_5 \times \sigma'(z_1) = -0.0498 \times 0.4 \times 0.1780 = -0.00354$$
$$\delta_2 = \delta_3 \times w_6 \times \sigma'(z_2) = -0.0498 \times 0.7 \times 0.0898 = -0.00313$$

**Step 4**：第一层权重梯度
$$\frac{\partial L}{\partial w_1} = \delta_1 \times x_1 = -0.00354 \times 1 = -0.00354$$
$$\frac{\partial L}{\partial w_2} = \delta_1 \times x_2 = -0.00354 \times 2 = -0.00708$$

## 7. 总结

反向传播的核心思想：

1. **链式法则**：将复杂求导分解为局部梯度连乘
2. **动态规划**：缓存中间结果避免重复计算
3. **反向传播**：从输出层向输入层逐层计算梯度
4. **梯度流动**：激活函数和权重初始化决定梯度是否消失/爆炸

> 💡 **记忆口诀**："前向算输出，反向传梯度，链式乘局部，梯度定更新"
