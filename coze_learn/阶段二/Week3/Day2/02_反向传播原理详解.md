# 反向传播原理详解

> **学习目标**：深入理解反向传播算法的数学本质，掌握链式法则在神经网络中的应用

---

## 1. 链式法则：反向传播的数学基础

### 1.1 导数回顾

对于函数 $y = f(x)$，导数定义为：
$$dy/dx = lim_ₑₗₜₐ ₓ ₜₒ ₀ (f(x + Δ x) - f(x))/(Δ x)$$

导数表示：当输入变化时，输出如何变化。

### 1.2 一维链式法则

对于复合函数 $y = f(g(x))$：
$$dy/dx = dy/dg · dg/dx$$

### 1.3 多元链式法则（核心）

对于 $z = f(u₁, u₂, ...)$，其中 $uᵢ = gᵢ(x)$：
$$(∂ z)/(∂ x) = Σᵢ (∂ z)/(∂ uᵢ) · (∂ uᵢ)/(∂ x)$$

**示例推导**：

给定：
- $z = u₁² + u₂ · u₁$
- $u₁ = 2x + 1$
- $u₂ = x²$

求解 $(∂ z)/(∂ x)$：

$$(∂ z)/(∂ u₁) = 2u₁ + u₂$$
$$(∂ z)/(∂ u₂) = u₁$$
$$(∂ u₁)/(∂ x) = 2$$
$$(∂ u₂)/(∂ x) = 2x$$

因此：
$$(∂ z)/(∂ x) = (2u₁ + u₂) · 2 + u₁ · 2x$$

## 2. 计算图：可视化神经网络

### 2.1 什么是计算图

计算图是神经网络的数学表示，每个节点代表一个操作，每条边代表数据流动。

```
输入 x ──→ [×w₁] ──→ [+] ──→ [σ] ──→ ŷ
            ↑          ↑
         权重 w₁    偏置 b
```

### 2.2 前向传播 vs 反向传播

| 阶段 | 方向 | 计算内容 |
|------|------|----------|
| 前向传播 | 左→右 | 计算输出 ${y}$ |
| 反向传播 | 右→左 | 计算梯度 $(∂ L)/(∂ w)$ |

### 2.3 典型MLP计算图

```
输入层        隐藏层        输出层
x₁ ──────┐
         ├──→ [Σ] ──→ [σ] ──┐
x₂ ──────┤   (z₁)          ├──→ [Σ] ──→ [σ] ──→ ŷ
         ├──→ [Σ] ──→ [σ] ──┘   (z₃)
x₃ ──────┘   (z₂)
```

## 3. 反向传播算法详解

### 3.1 单层神经网络的梯度计算

**前向传播**：
$$z = w · x + b$$
$${y} = σ(z)$$

**损失函数**（MSE）：
$$L = 1/2(y - {y})²$$

**反向传播推导**：

**Step 1**：输出层梯度
$${∂ L}{∂ {y}} = {y} - y$$

**Step 2**：激活函数梯度
$${∂ {y}}{∂ z} = σ'(z) = σ(z)(1 - σ(z))$$

**Step 3**：加权求和梯度
$$(∂ z)/(∂ wᵢ) = xᵢ, (∂ z)/(∂ b) = 1$$

**Step 4**：综合梯度（链式法则）
$$(∂ L)/(∂ wᵢ) = {∂ L}{∂ {y}} · {∂ {y}}{∂ z} · (∂ z)/(∂ wᵢ)$$

展开：
$$(∂ L)/(∂ wᵢ) = ({y} - y) · σ'(z) · xᵢ$$

### 3.2 多层神经网络的梯度计算

对于 $L$ 层网络：

```
输入 x → [Layer₁] → [Layer₂] → ... → [Layerₗ] → ... → [Layerᴸ] → 输出
              ↑            ↑                    ↑
            梯度δ¹       梯度δ²                梯度δᴸ
```

**通用递推公式**：

第 $l$ 层的梯度 $δˡ$：
$$δˡ = (∂ L)/(∂ zˡ)$$

**从后向前传播**：
$$δˡ = (Wˡ⁺¹)ᵀ δˡ⁺¹ σ'(zˡ)$$

其中 $$表示逐元素乘法。 --- ## 4. 梯度消失与梯度爆炸 ### 4.1 梯度消失（Vanishing Gradient） **问题描述**：$$∂ L/∂ w⁽¹⁾ = {∂ L}{∂ {y}} · ∏_ᵢ₌₂L σ'(zⁱ) · ∏_ᵢ₌₁L⁻¹ Wⁱ$$

当 $|Wσ'(z)| < 1$ 时，梯度逐层指数衰减。

**影响**：
- 靠近输入层的权重几乎不更新
- 网络难以学习深层特征
- Sigmoid/tanh 激活函数尤其容易出现

### 4.2 梯度爆炸（Exploding Gradient）

**问题描述**：
$$∂ L/∂ w⁽¹⁾ = {∂ L}{∂ {y}} · ∏_ᵢ₌₂L σ'(zⁱ) · ∏_ᵢ₌₁L⁻¹ Wⁱ$$

当 $|Wσ'(z)| > 1$ 时，梯度指数增长。

**影响**：
- 参数更新过大
- 损失函数震荡/发散
- 数值溢出（NaN）

### 4.3 对比表格

| 现象 | 梯度消失 | 梯度爆炸 |
|------|----------|----------|
| 梯度值 | → 0 | → ∞ |
| 训练表现 | 前层不学习 | 震荡/发散 |
| 常见场景 | 深层网络+Sigmoid | 深层网络+大权重 |
| 解决方案 | ReLU/残差连接 | 梯度裁剪/LSTM |

## 5. 激活函数对梯度的影响

### 5.1 Sigmoid

$$σ(x) = 1/(1 + e⁻ˣ)$$

$$σ'(x) = σ(x)(1 - σ(x))$$

**特点**：
- 输出范围：(0, 1)
- 最大导数：0.25（当 x=0 时）
- 深层网络梯度快速衰减

**梯度曲线**：
```
导数值
  0.25 ┤╮
       │ ╰─→ 最大值
  0.00 ┤──────────╯
       └──────────────→ x
        -10  0  10
```

### 5.2 Tanh

$$(x) = (eˣ - e⁻ˣ)/(eˣ + e⁻ˣ)$$

$$'(x) = 1 - ²(x)$$

**特点**：
- 输出范围：(-1, 1)
- 最大导数：1
- 零中心化，但仍有梯度消失

### 5.3 ReLU

$$f(x) = max(0, x)$$

$$f'(x) = {cases} 1 & x > 0 0 & x ≤ 0 {cases}$$

**特点**：
- 缓解梯度消失（正区间梯度恒为1）
- 计算高效
- **dying ReLU**：负区间神经元可能永久死亡

### 5.4 梯度特性对比

| 激活函数 | 梯度范围 | 零中心 | 梯度消失风险 |
|----------|----------|--------|--------------|
| Sigmoid | (0, 0.25] | 否 | 高 |
| Tanh | (0, 1] | 是 | 中 |
| ReLU | {0, 1} | 否 | 低 |
| Leaky ReLU | (α, 1] | 否 | 低 |
| ELU | (α, 1] | 是 | 低 |

## 6. 实战：手推反向传播

### 6.1 示例网络

```
输入: x₁=1, x₂=2
权重: w₁=0.5, w₂=0.3, w₃=0.8, w₄=0.6
偏置: b₁=0.1, b₂=0.2
激活函数: Sigmoid
损失: MSE, 真实值 y=1
```

### 6.2 前向传播

**第一层**：
$$z₁ = w₁ x₁ + w₂ x₂ + b₁ = 0.5 × 1 + 0.3 × 2 + 0.1 = 1.2$$
$$a₁ = σ(z₁) = 1/(1+e⁻¹.²) ≈ 0.7685$$

$$z₂ = w₃ x₁ + w₄ x₂ + b₂ = 0.8 × 1 + 0.6 × 2 + 0.2 = 2.2$$
$$a₂ = σ(z₂) = 1/(1+e⁻².²) ≈ 0.9002$$

**第二层**（输出层，假设单输出）：
$$z₃ = w₅ a₁ + w₆ a₂ + b₃ = 0.4 × 0.7685 + 0.7 × 0.9002 + 0.1 = 1.047$$
$${y} = σ(z₃) = 1/(1+e⁻¹.⁰⁴⁷) ≈ 0.7400$$

**损失**：
$$L = 1/2(y - {y})² = 1/2(1 - 0.74)² = 0.0338$$

### 6.3 反向传播

**Step 1**：输出层
$$δ₃ = (∂ L)/(∂ z₃) = (0.74 - 1) × 0.74 × 0.26 = -0.0498$$

**Step 2**：第二层权重梯度
$$(∂ L)/(∂ w₅) = δ₃ × a₁ = -0.0498 × 0.7685 = -0.0383$$
$$(∂ L)/(∂ w₆) = δ₃ × a₂ = -0.0498 × 0.9002 = -0.0448$$

**Step 3**：传播到第一层
$$δ₁ = δ₃ × w₅ × σ'(z₁) = -0.0498 × 0.4 × 0.1780 = -0.00354$$
$$δ₂ = δ₃ × w₆ × σ'(z₂) = -0.0498 × 0.7 × 0.0898 = -0.00313$$

**Step 4**：第一层权重梯度
$$(∂ L)/(∂ w₁) = δ₁ × x₁ = -0.00354 × 1 = -0.00354$$
$$(∂ L)/(∂ w₂) = δ₁ × x₂ = -0.00354 × 2 = -0.00708$$

## 7. 总结

反向传播的核心思想：

1. **链式法则**：将复杂求导分解为局部梯度连乘
2. **动态规划**：缓存中间结果避免重复计算
3. **反向传播**：从输出层向输入层逐层计算梯度
4. **梯度流动**：激活函数和权重初始化决定梯度是否消失/爆炸

> 💡 **记忆口诀**："前向算输出，反向传梯度，链式乘局部，梯度定更新"
