## 📚 学习内容清单

### Part 1: 端到端项目实战

- [ ] **打开 Jupyter Notebook**: `图像分类端到端项目.ipynb`

#### 1.1 数据准备与增强 (13:00-14:00)
- [ ] 加载 CIFAR-10 数据集
- [ ] 实现数据增强策略
  - [ ] 随机裁剪 (RandomCrop)
  - [ ] 水平翻转 (HorizontalFlip)
  - [ ] 颜色抖动 (ColorJitter)
  - [ ] 随机旋转 (RandomRotation)
- [ ] 可视化数据样本

#### 1.2 模型设计与实现 (14:00-14:30)
- [ ] 理解 CNN 基础结构
  - [ ] 卷积层 (Conv2d)
  - [ ] 批归一化 (BatchNorm2d)
  - [ ] 激活函数 (ReLU)
  - [ ] 池化层 (MaxPool2d)
  - [ ] 全连接层 (Linear)
- [ ] 实现 BasicCNN 模型
- [ ] 实现迁移学习模型 (TransferResNet)
- [ ] 对比两种模型结构

#### 1.3 训练与调参 (14:30-15:30)
- [ ] 配置训练参数
  - [ ] 学习率 (Learning Rate)
  - [ ] Batch Size
  - [ ] 优化器选择 (SGD/Adam/AdamW)
- [ ] 实现训练循环
- [ ] 监控训练过程
  - [ ] 训练损失
  - [ ] 验证损失
  - [ ] 准确率
- [ ] 保存最佳模型

### Part 2: 模型评估与错误分析

#### 2.1 评估指标 (15:30-16:00)
- [ ] 计算测试集准确率
- [ ] 生成混淆矩阵
- [ ] 分析分类报告 (Precision/Recall/F1)

#### 2.2 错误分析 (16:00-16:30)
- [ ] 识别误分类样本
- [ ] 分析错误模式
  - [ ] 哪些类别容易混淆？
  - [ ] 错误样本有什么特点？
- [ ] 计算各类别准确率

### Part 3: 优化技术学习

#### 3.1 学习率调度
- [ ] 阅读 `项目优化指南.md` 第1节
- [ ] 理解各种调度策略：
  - [ ] StepLR (阶梯衰减)
  - [ ] CosineAnnealingLR (余弦退火)
  - [ ] ReduceLROnPlateau (自适应衰减)
- [ ] 在项目中实践学习率调度

#### 3.2 正则化技术
- [ ] 阅读 `项目优化指南.md` 第2节
- [ ] 理解正则化方法：
  - [ ] L2 正则化 (Weight Decay)
  - [ ] Dropout
  - [ ] Batch Normalization
  - [ ] Label Smoothing
- [ ] 对比有/无正则化的效果

#### 3.3 早停策略
- [ ] 阅读 `项目优化指南.md` 第3节
- [ ] 实现 EarlyStopping
- [ ] 理解 patience 参数的作用

#### 3.4 超参数搜索
- [ ] 阅读 `项目优化指南.md` 第4节
- [ ] 理解搜索策略：
  - [ ] Grid Search (网格搜索)
  - [ ] Random Search (随机搜索)
  - [ ] Bayesian Optimization (贝叶斯优化)

### Part 4: 模型部署入门

#### 4.1 ONNX 导出
- [ ] 阅读 `模型部署入门.md` 第2-3节
- [ ] 理解 ONNX 格式
- [ ] 学习 torch.onnx.export 用法

#### 4.2 ONNX Runtime 推理
- [ ] 阅读 `模型部署入门.md` 第4节
- [ ] 安装 onnxruntime
- [ ] 实现简单的 ONNX 推理

#### 4.3 模型优化
- [ ] 了解模型量化
- [ ] 了解模型剪枝
- [ ] 了解知识蒸馏
