## 1. 学习率调度 <a name="1-学习率调度"></a>

### 1.1 为什么需要学习率调度？

学习率是最重要的超参数之一：
- **过大**：训练不稳定，可能发散
- **过小**：收敛太慢，容易陷入局部最优
- **动态调整**：初期大学习率快速探索，后期小学习率精细收敛

### 1.2 常见学习率调度策略

#### StepLR（阶梯衰减）

```python
import torch.optim as optim

optimizer = optim.SGD(model.parameters(), lr=0.1, momentum=0.9)
scheduler = optim.lr_scheduler.StepLR(optimizer, step_size=30, gamma=0.1)

# 每30个epoch学习率乘以0.1
# 0.1 -> 0.01 -> 0.001 -> 0.0001
```

**适用场景**：有明确阶段性特征的任务

#### CosineAnnealingLR（余弦退火）

```python
scheduler = optim.lr_scheduler.CosineAnnealingLR(
    optimizer, 
    T_max=100,           # 最大迭代周期
    eta_min=1e-6          # 最小学习率
)
```

**特点**：平滑下降，理论上有更好的收敛性

#### ReduceLROnPlateau（ plateau  plateau  探测）

```python
scheduler = optim.lr_scheduler.ReduceLROnPlateau(
    optimizer,
    mode='min',           # 监控指标：min(损失) 或 max(准确率)
    factor=0.5,           # 衰减因子
    patience=10,          # 耐心值
    verbose=True          # 打印学习率变化
)
```

**适用场景**：通用性强，自动适应训练状态

#### Warmup + Cosine（预热+余弦）

```python
class WarmupCosineScheduler:
    def __init__(self, optimizer, warmup_epochs, total_epochs, 
                 base_lr, min_lr=0):
        self.optimizer = optimizer
        self.warmup_epochs = warmup_epochs
        self.total_epochs = total_epochs
        self.base_lr = base_lr
        self.min_lr = min_lr
        
    def step(self, epoch):
        if epoch < self.warmup_epochs:
            lr = self.base_lr * (epoch + 1) / self.warmup_epochs
        else:
            progress = (epoch - self.warmup_epochs) / (self.total_epochs - self.warmup_epochs)
            lr = self.min_lr + 0.5 * (self.base_lr - self.min_lr) * (1 + np.cos(np.pi * progress))
        
        for param_group in self.optimizer.param_groups:
            param_group['lr'] = lr
        return lr
```

### 1.3 学习率调度对比实验

| 策略 | 收敛速度 | 最终效果 | 适用场景 |
|------|----------|----------|----------|
| StepLR | ⭐⭐ | ⭐⭐⭐ | 简单任务 |
| CosineAnnealing | ⭐⭐⭐ | ⭐⭐⭐ | 通用场景 |
| ReduceLROnPlateau | ⭐⭐⭐ | ⭐⭐⭐⭐ | 训练不稳定 |
| Warmup+Cosine | ⭐⭐⭐⭐ | ⭐⭐⭐⭐ | 大模型训练 |

## 2. 正则化技术 <a name="2-正则化技术"></a>

### 2.1 什么是正则化？

正则化防止过拟合，让模型在训练集和测试集上都能表现良好。

### 2.2 L1 和 L2 正则化

```python
# L2正则化 (Weight Decay)
optimizer = optim.Adam(
    model.parameters(),
    lr=0.001,
    weight_decay=1e-4  # L2正则化强度
)

# L1正则化需要在损失函数中添加
l1_loss = 0.0
for param in model.parameters():
    l1_loss += torch.abs(param).sum()
loss = ce_loss + 0.001 * l1_loss
```

| 类型 | 特点 | 效果 |
|------|------|------|
| L2 | 权重衰减 | 使权重接近0但不完全为0 |
| L1 | 稀疏化 | 产生真正的零权重（特征选择） |

### 2.3 Dropout

```python
class DropoutCNN(nn.Module):
    def __init__(self):
        super().__init__()
        self.conv1 = nn.Conv2d(3, 64, 3, padding=1)
        self.dropout1 = nn.Dropout2d(0.25)  # 卷积层Dropout
        self.fc1 = nn.Linear(512, 256)
        self.dropout2 = nn.Dropout(0.5)     # 全连接层Dropout
        
    def forward(self, x):
        x = F.relu(self.conv1(x))
        x = self.dropout1(x)
        x = F.max_pool2d(x, 2)
        x = x.view(x.size(0), -1)
        x = F.relu(self.fc1(x))
        x = self.dropout2(x)
        x = self.fc2(x)
        return x
```

**使用技巧**：
- 卷积层：用 `Dropout2d(p=0.25)`
- 全连接层：用 `Dropout(p=0.5)`
- 测试时自动关闭

### 2.4 Batch Normalization

```python
class BNConvNet(nn.Module):
    def __init__(self):
        super().__init__()
        self.features = nn.Sequential(
            nn.Conv2d(3, 64, 3, padding=1),
            nn.BatchNorm2d(64),    # 批归一化
            nn.ReLU(inplace=True),
            nn.MaxPool2d(2, 2)
        )
```

**原理**：
- 标准化层输入：$\hat{x} = \frac{x - \mu}{\sqrt{\sigma^2 + \epsilon}}$
- 可学习参数：$\gamma, \beta$
- 输出：$y = \gamma \hat{x} + \beta$

### 2.5 Label Smoothing（标签平滑）

```python
class LabelSmoothingCrossEntropy(nn.Module):
    def __init__(self, smoothing=0.1):
        super().__init__()
        self.smoothing = smoothing
        
    def forward(self, pred, target):
        n_classes = pred.size(1)
        confidence = 1.0 - self.smoothing
        smooth_value = self.smoothing / (n_classes - 1)
        
        # one-hot编码
        one_hot = torch.zeros_like(pred).scatter(1, target.unsqueeze(1), 1)
        # 平滑
        smooth_target = one_hot * confidence + (1 - one_hot) * smooth_value
        
        # 交叉熵
        log_prob = F.log_softmax(pred, dim=1)
        loss = (-smooth_target * log_prob).sum(dim=1).mean()
        return loss
```

**效果**：防止模型对训练样本过度自信，提升泛化能力

### 2.6 正则化技术对比

| 技术 | 作用 | 强度控制 | 适用场景 |
|------|------|----------|----------|
| L2 正则化 | 权重衰减 | weight_decay | 通用 |
| Dropout | 神经元随机失活 | p 值 | 全连接层 |
| BatchNorm | 层输入归一化 | 无需调参 | 卷积层 |
| Label Smoothing | 软化标签 | smoothing | 分类任务 |

## 3. 早停策略 <a name="3-早停策略"></a>

### 3.1 为什么需要早停？

- 防止过拟合：验证集性能下降时停止训练
- 节省时间：不在无意义的训练上浪费时间
- 保存最佳模型：保留验证集上最好的模型

### 3.2 早停实现

```python
class EarlyStopping:
    def __init__(self, patience=10, min_delta=0.001, mode='min', verbose=True):
        self.patience = patience          # 耐心值
        self.min_delta = min_delta         # 最小改善阈值
        self.mode = mode                   # 'min' 或 'max'
        self.verbose = verbose
        self.counter = 0
        self.best_score = None
        self.early_stop = False
        self.best_epoch = 0
        
    def __call__(self, epoch, score):
        """
        检查是否应该早停
        epoch: 当前轮次
        score: 当前指标值（验证损失或准确率）
        """
        if self.best_score is None:
            self.best_score = score
            self.best_epoch = epoch
            
        elif self._is_improvement(score):
            self.best_score = score
            self.best_epoch = epoch
            self.counter = 0  # 重置计数器
            
        else:
            self.counter += 1
            if self.verbose:
                print(f'  No improvement. Patience: {self.counter}/{self.patience}')
                
            if self.counter >= self.patience:
                self.early_stop = True
                if self.verbose:
                    print(f'\n🛑 Early stopping at epoch {epoch+1}')
                    print(f'   Best epoch: {self.best_epoch+1}, Best score: {self.best_score:.4f}')
                    
    def _is_improvement(self, score):
        if self.mode == 'min':
            return score < self.best_score - self.min_delta
        else:
            return score > self.best_score + self.min_delta
```

### 3.3 早停参数选择

```
patience 设置建议：
├── 快速迭代：patience = 5-10
├── 标准训练：patience = 10-20  
└── 大模型/长训练：patience = 20-30

min_delta 设置建议：
├── 稳定指标：min_delta = 0.001
├── 波动指标：min_delta = 0.01
└── 精确控制：min_delta = 0.0001
```

### 3.4 高级早停策略

```python
class AdvancedEarlyStopping:
    """带恢复功能的早停"""
    
    def __init__(self, patience=15, min_delta=0.001, 
                 restore_best=True, cooldown=5):
        self.patience = patience
        self.min_delta = min_delta
        self.restore_best = restore_best
        self.cooldown = cooldown
        self.counter = 0
        self.best_score = None
        self.early_stop = False
        self.cooldown_counter = 0
        
    def __call__(self, epoch, score, model, optimizer):
        # 冷却期检查
        if self.cooldown_counter > 0:
            self.cooldown_counter -= 1
            return
            
        if self.best_score is None:
            self.best_score = score
            self._save_checkpoint(epoch, model, optimizer)
            
        elif score < self.best_score - self.min_delta:
            self.best_score = score
            self.counter = 0
            self._save_checkpoint(epoch, model, optimizer)
            
        else:
            self.counter += 1
            if self.counter >= self.patience:
                self.early_stop = True
                if self.restore_best:
                    self._restore_best(model, optimizer)
                    
    def _save_checkpoint(self, epoch, model, optimizer):
        torch.save({
            'epoch': epoch,
            'model_state_dict': model.state_dict(),
            'optimizer_state_dict': optimizer.state_dict(),
        }, 'checkpoint_best.pth')
```

## 4. 超参数搜索 <a name="4-超参数搜索"></a>

### 4.1 网格搜索 (Grid Search)

```python
from itertools import product

def grid_search():
    """网格搜索最佳超参数"""
    
    # 定义搜索空间
    learning_rates = [0.001, 0.01, 0.1]
    batch_sizes = [32, 64, 128]
    weight_decays = [1e-4, 1e-3, 1e-2]
    
    results = []
    
    # 遍历所有组合
    for lr, bs, wd in product(learning_rates, batch_sizes, weight_decays):
        print(f'Testing: lr={lr}, batch_size={bs}, weight_decay={wd}')
        
        # 训练模型
        model = build_model()
        train_loader, val_loader = get_data_loaders(batch_size=bs)
        optimizer = optim.Adam(model.parameters(), lr=lr, weight_decay=wd)
        
        # 训练和评估
        val_acc = train_and_evaluate(model, optimizer, train_loader, val_loader)
        
        results.append({
            'lr': lr,
            'batch_size': bs,
            'weight_decay': wd,
            'val_acc': val_acc
        })
        
    # 返回最佳配置
    best = max(results, key=lambda x: x['val_acc'])
    return best
```

**缺点**：组合爆炸，效率低

### 4.2 随机搜索 (Random Search)

```python
import random

def random_search(n_trials=50):
    """随机搜索超参数"""
    
    results = []
    
    for i in range(n_trials):
        # 从分布中随机采样
        lr = random.uniform(1e-5, 1e-1)
        bs = random.choice([16, 32, 64, 128, 256])
        wd = random.uniform(1e-6, 1e-2)
        dropout = random.uniform(0.1, 0.7)
        
        print(f'Trial {i+1}/{n_trials}: lr={lr:.6f}, bs={bs}, wd={wd:.6f}')
        
        # 训练和评估
        model = build_model()
        optimizer = optim.Adam(model.parameters(), lr=lr, weight_decay=wd)
        val_acc = train_and_evaluate(model, optimizer, train_loader, val_loader)
        
        results.append({
            'lr': lr, 'batch_size': bs, 
            'weight_decay': wd, 'dropout': dropout,
            'val_acc': val_acc
        })
        
    return max(results, key=lambda x: x['val_acc'])
```

**优势**：在相同预算下通常优于网格搜索

### 4.3 贝叶斯优化 (Bayesian Optimization)

```python
# 使用 Optuna 进行贝叶斯优化
import optuna

def objective(trial):
    """Optuna 目标函数"""
    
    # 定义搜索空间
    lr = trial.suggest_float('learning_rate', 1e-5, 1e-1, log=True)
    bs = trial.suggest_categorical('batch_size', [32, 64, 128, 256])
    wd = trial.suggest_float('weight_decay', 1e-6, 1e-2, log=True)
    num_layers = trial.suggest_int('num_layers', 2, 6)
    hidden_dim = trial.suggest_categorical('hidden_dim', [64, 128, 256, 512])
    
    # 训练
    model = build_model(layers=num_layers, hidden=hidden_dim)
    optimizer = optim.Adam(model.parameters(), lr=lr, weight_decay=wd)
    val_acc = train_and_evaluate(model, optimizer, train_loader, val_loader, bs)
    
    return val_acc

# 运行优化
study = optuna.create_study(direction='maximize')
study.optimize(objective, n_trials=100)

print(f'Best trial: {study.best_trial.params}')
print(f'Best accuracy: {study.best_trial.value:.4f}')
```

### 5.1 实验记录模板

```python
class ExperimentTracker:
    def __init__(self, experiment_name):
        self.experiment_name = experiment_name
        self.results = []
        self.start_time = time.time()
        
    def log_config(self, config):
        """记录实验配置"""
        self.config = config
        print(f"\n{'='*60}")
        print(f"实验: {self.experiment_name}")
        print(f"{'='*60}")
        print("配置:")
        for k, v in config.items():
            print(f"  {k}: {v}")
            
    def log_metrics(self, epoch, metrics):
        """记录每个epoch的指标"""
        self.results.append({
            'epoch': epoch,
            **metrics,
            'time': time.time() - self.start_time
        })
        
    def log_summary(self):
        """生成实验总结"""
        df = pd.DataFrame(self.results)
        best_idx = df['val_acc'].idxmax()
        best = df.iloc[best_idx]
        
        summary = f"""
{'='*60}
实验总结: {self.experiment_name}
{'='*60}
最佳验证准确率: {best['val_acc']:.4f}% (Epoch {int(best['epoch'])})        
最终训练准确率: {df.iloc[-1]['train_acc']:.4f}%
最佳验证损失: {best['val_loss']:.4f}
总训练时间: {df.iloc[-1]['time']:.1f}秒
过拟合程度: {df.iloc[-1]['train_acc'] - best['val_acc']:.4f}%
"""
        print(summary)
        return df
```

## 📝 总结

### 核心优化技术速查表

| 技术 | 作用 | 关键参数 |
|------|------|----------|
| 学习率调度 | 平衡收敛速度与精度 | 初始学习率、衰减策略 |
| L2 正则化 | 防止过拟合 | weight_decay (1e-4 ~ 1e-2) |
| Dropout | 增强泛化 | p (0.2 ~ 0.5) |
| BatchNorm | 稳定训练 | 无需调参 |
| 早停 | 防止过拟合 | patience (10 ~ 20) |
| Label Smoothing | 防止过度自信 | smoothing (0.1) |

### 优化建议流程

```
1. ✅ Baseline: 简单模型 + 标准配置
     ↓
2. ✅ + BatchNorm + 基础正则化
     ↓
3. ✅ + 学习率调度 (CosineAnnealing)
     ↓
4. ✅ + 早停策略
     ↓
5. ✅ 超参数微调 (Bayesian Optimization)
     ↓
6. ✅ 高级技术 (MixUp, CutMix, SWA...)
```
