### 🔑 核心概念速查

```
┌─────────────────────────────────────────────────────────────┐
│                        CNN 核心概念                          │
├─────────────────────────────────────────────────────────────┤
│                                                             │
│  卷积操作：滑动窗口 + 权重共享 → 提取空间特征                │
│                                                             │
│  池化操作：降维 + 增强鲁棒性                                 │
│    - Max Pooling：保留显著特征                              │
│    - Avg Pooling：保留背景信息                              │
│                                                             │
│  感受野：输出像素对应的输入区域范围                           │
│    - 浅层 → 小感受野 → 边缘/纹理                            │
│    - 深层 → 大感受野 → 语义/物体                           │
│                                                             │
│  架构演进：                                                  │
│    LeNet(1998) → AlexNet(2012) → VGG(2014) → ResNet(2015)  │
│      开创        突破        简约         里程碑            │
│                                                             │
│  残差连接：y = F(x) + x  → 解决深层网络训练难题              │
│                                                             │
└─────────────────────────────────────────────────────────────┘
```

### ⚙️ 常见问题排查

| 问题 | 可能原因 | 解决方案 |
|------|----------|----------|
| CUDA out of memory | 批量太大 | 减小 BATCH_SIZE |
| 训练loss不下降 | 学习率不合适 | 调小学习率 |
| 准确率很低 | 数据增强过度 | 减弱增强 |
| 模型加载失败 | 路径错误 | 检查文件路径 |
