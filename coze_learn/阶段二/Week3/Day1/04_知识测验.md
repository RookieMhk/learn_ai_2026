## 第一题：感知机与逻辑门

**问题**：以下哪种逻辑门**无法**用单层感知机实现？

```
A. AND (与门)
B. OR  (或门)
C. NOT (非门)
D. XOR (异或门)
```

<details>
<summary>💡 提示</summary>

单层感知机只能实现**线性可分**的分类问题。

</details>

## 第二题：激活函数特性

**问题**：关于 ReLU 激活函数的说法，正确的是？

```
A. ReLU 的输出范围是 (0, 1)
B. ReLU 在 x < 0 时梯度为 0，可能导致"死神经元"问题
C. ReLU 的计算需要指数运算，速度较慢
D. ReLU 只能用于输出层，不能用于隐藏层
```

<details>
<summary>💡 提示</summary>

ReLU = max(0, x)，是一种简单高效的激活函数。

</details>

## 第三题：MLP 前向传播

**问题**：某 MLP 包含输入层(784神经元)、隐藏层1(256神经元)、隐藏层2(128神经元)、输出层(10神经元)。忽略偏置项，隐藏层1的权重矩阵形状是？

```
A. (784, 256)
B. (256, 784)
C. (784, 128)
D. (256, 128)
```

<details>
<summary>💡 提示</summary>

权重矩阵形状为 **(输出维度, 输入维度)**

</details>

## 第四题：激活函数选择

**问题**：在构建一个用于 ImageNet 分类的深度卷积神经网络时，以下哪个是隐藏层的**最佳默认激活函数**选择？

```
A. Sigmoid
B. Tanh  
C. ReLU
D. Softmax
```

## 第五题：前向传播计算

**问题**：给定感知机参数 w₁=0.5, w₂=-0.5, b=0，激活函数为 sign(>0 → 1, ≤0 → 0)。对于输入 x₁=1, x₂=1，输出是？

```
A. 0
B. 1
C. -1
D. 0.5
```

<details>
<summary>💡 提示</summary>

计算：y = w₁×x₁ + w₂×x₂ + b = ?

</details>

---

## 答案与解析

### 第一题答案：D (XOR)

**解析**：
- AND、OR、NOT 都是线性可分的，可以用单层感知机实现
- XOR 是非线性可分问题，在二维空间中不存在一条直线能将 (0,0)和(1,1)与(0,1)和(1,0)分开
- 解决 XOR 需要至少一个**隐藏层**的多层感知机

### 第二题答案：B

**解析**：
- A 错误：ReLU 输出范围是 [0, +∞)，不是 (0, 1)
- B **正确**：当 x < 0 时，ReLU 输出 0，梯度也是 0，导致这些神经元"死亡"
- C 错误：ReLU 只涉及 max 比较，计算高效
- D 错误：ReLU 是隐藏层的**首选**激活函数

### 第三题答案：B (256, 784)

**解析**：
- 隐藏层1接收 784 维输入，输出 256 维
- 权重矩阵形状 = (输出维度, 输入维度) = **(256, 784)**
- 输入层 → 隐藏层1：Z¹ = W¹·X + b¹，其中 W¹ ∈ ℝ^(256×784)

### 第四题答案：C (ReLU)

**解析**：
- A Sigmoid：梯度消失严重，不适合深度网络
- B Tanh：比 Sigmoid 好（零均值），但仍有梯度消失
- C **ReLU**：深度学习的**默认选择**，计算高效、梯度不消失
- D Softmax：用于**多分类输出层**，不是隐藏层

### 第五题答案：A (0)

**解析**：
- 计算线性组合：y = 0.5×1 + (-0.5)×1 + 0 = 0.5 - 0.5 = 0
- 激活函数 sign：0 既不大于 0，也不小于等于 0，按约定输出 0
- 因此最终输出为 **0**
