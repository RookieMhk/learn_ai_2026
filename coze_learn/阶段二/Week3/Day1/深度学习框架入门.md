## 2. PyTorch 核心概念

### 2.1 张量 (Tensor)

**张量 = 多维数组 + GPU 加速 + 自动求导**

```python
import torch

# 创建张量的多种方式
a = torch.tensor([1, 2, 3])           # 从列表
b = torch.zeros(3, 4)                  # 全零矩阵
c = torch.ones(2, 3)                   # 全一矩阵
d = torch.randn(4, 4)                  # 标准正态分布
e = torch.arange(0, 10, step=2)        # 序列
f = torch.linspace(0, 1, steps=5)      # 线性间隔
g = torch.eye(3)                       # 单位矩阵

# 指定设备 (CPU/GPU)
x_gpu = torch.tensor([1, 2, 3], device='cuda')
x_mps = torch.tensor([1, 2, 3], device='mps')  # Apple Silicon

# 指定数据类型
x_float = torch.ones(3, dtype=torch.float32)
x_int = torch.zeros(5, dtype=torch.int64)
```

### 2.2 自动求导 (Autograd)

PyTorch 的核心特性：**自动计算梯度**

```python
# 创建需要梯度的张量
x = torch.tensor([2.0, 3.0], requires_grad=True)

# 定义计算图
y = x ** 2 + 2 * x + 1  # y = x² + 2x + 1

# 自动求导
y.sum().backward()  # 计算 dy/dx

# 查看梯度
print(x.grad)  # tensor([6., 8.])
# dy/dx = 2x + 2 → [2*2+2, 2*3+2] = [6, 8]
```

### 2.3 神经网络模块 (nn.Module)

所有神经网络模型的基类：

```python
import torch.nn as nn

class SimpleNet(nn.Module):
    def __init__(self, input_size, hidden_size, output_size):
        super().__init__()
        self.layer1 = nn.Linear(input_size, hidden_size)
        self.relu = nn.ReLU()
        self.layer2 = nn.Linear(hidden_size, output_size)
    
    def forward(self, x):
        x = self.layer1(x)
        x = self.relu(x)
        x = self.layer2(x)
        return x

# 使用模型
model = SimpleNet(784, 256, 10)
output = model(input_tensor)
```

## 3. TensorFlow 核心概念

### 3.1 Keras API (推荐)

TensorFlow 2.x 的高层 API：

```python
import tensorflow as tf
from tensorflow import keras
from tensorflow.keras import layers

# 方式1：Sequential API (适合简单层叠)
model = keras.Sequential([
    layers.Dense(256, activation='relu', input_shape=(784,)),
    layers.Dropout(0.3),
    layers.Dense(128, activation='relu'),
    layers.Dense(10, activation='softmax')
])

# 方式2：函数式 API (适合复杂拓扑)
inputs = keras.Input(shape=(784,))
x = layers.Dense(256, activation='relu')(inputs)
x = layers.Dropout(0.3)(x)
x = layers.Dense(128, activation='relu')(x)
outputs = layers.Dense(10, activation='softmax')(x)
model = keras.Model(inputs=inputs, outputs=outputs)

# 方式3：自定义模型类
class MyModel(keras.Model):
    def __init__(self):
        super().__init__()
        self.dense1 = layers.Dense(256, activation='relu')
        self.dense2 = layers.Dense(128, activation='relu')
        self.output_layer = layers.Dense(10, activation='softmax')
    
    def call(self, inputs):
        x = self.dense1(inputs)
        x = self.dense2(x)
        return self.output_layer(x)
```

## 4. 张量(Tensor)操作详解

### 4.1 核心操作对比

| 操作 | PyTorch | TensorFlow |
|:-----|:--------|:-----------|
| 创建 | `torch.tensor([1,2])` | `tf.constant([1,2])` |
| 形状 | `x.shape` | `x.shape` |
| 重塑 | `x.reshape(2,3)` | `tf.reshape(x, [2,3])` |
| 切片 | `x[0:2, 1:]` | `x[0:2, 1:]` |
| 拼接 | `torch.cat([a,b])` | `tf.concat([a,b], axis=0)` |
| 堆叠 | `torch.stack([a,b])` | `tf.stack([a,b])` |
| 矩阵乘 | `torch.mm(a,b)` / `@` | `tf.matmul(a,b)` |
| 元素乘 | `a * b` | `a * b` |
| 求和 | `x.sum()` | `tf.reduce_sum(x)` |
| 均值 | `x.mean()` | `tf.reduce_mean(x)` |

### 4.2 NumPy 互转

**PyTorch ↔ NumPy：**
```python
import numpy as np
import torch

# NumPy → PyTorch
np_array = np.array([1, 2, 3])
torch_tensor = torch.from_numpy(np_array)
torch_tensor = torch.tensor(np_array)  # 复制数据

# PyTorch → NumPy
back_to_np = torch_tensor.numpy()

# ⚠️ 注意事项：共享内存时注意 detach()
torch_tensor = torch_tensor.detach()  # 分离计算图
```

**TensorFlow ↔ NumPy：**
```python
import numpy as np
import tensorflow as tf

# NumPy → TensorFlow
np_array = np.array([1, 2, 3])
tf_tensor = tf.constant(np_array)

# TensorFlow → NumPy
back_to_np = tf_tensor.numpy()
```

### 5.2 PyTorch 完整示例 (MNIST 分类)

```python
import torch
import torch.nn as nn
import torch.optim as optim
from torch.utils.data import DataLoader, TensorDataset

# 1. 准备数据 (模拟)
x_train = torch.randn(1000, 784)
y_train = torch.randint(0, 10, (1000,))
train_dataset = TensorDataset(x_train, y_train)
train_loader = DataLoader(train_dataset, batch_size=32, shuffle=True)

# 2. 定义模型
class MLP(nn.Module):
    def __init__(self):
        super().__init__()
        self.layers = nn.Sequential(
            nn.Linear(784, 256),
            nn.ReLU(),
            nn.Dropout(0.2),
            nn.Linear(256, 128),
            nn.ReLU(),
            nn.Linear(128, 10)
        )
    
    def forward(self, x):
        return self.layers(x)

model = MLP()

# 3. 损失函数和优化器
criterion = nn.CrossEntropyLoss()
optimizer = optim.Adam(model.parameters(), lr=0.001)

# 4. 训练循环
for epoch in range(5):
    total_loss = 0
    for batch_x, batch_y in train_loader:
        # 前向传播
        outputs = model(batch_x)
        loss = criterion(outputs, batch_y)
        
        # 反向传播
        optimizer.zero_grad()
        loss.backward()
        optimizer.step()
        
        total_loss += loss.item()
    
    print(f"Epoch {epoch+1}, Loss: {total_loss/len(train_loader):.4f}")

# 5. 预测
model.eval()
with torch.no_grad():
    test_input = torch.randn(1, 784)
    prediction = model(test_input)
    print(f"预测类别: {prediction.argmax(1).item()}")
```

### 5.3 TensorFlow 完整示例 (MNIST 分类)

```python
import tensorflow as tf
from tensorflow.keras import layers, models

# 1. 定义模型
model = models.Sequential([
    layers.Dense(256, activation='relu', input_shape=(784,)),
    layers.Dropout(0.2),
    layers.Dense(128, activation='relu'),
    layers.Dense(10, activation='softmax')
])

# 2. 编译
model.compile(
    optimizer='adam',
    loss='sparse_categorical_crossentropy',
    metrics=['accuracy']
)

# 3. 训练 (使用内置数据集)
(x_train, y_train), (x_test, y_test) = tf.keras.datasets.mnist.load_data()
x_train = x_train.reshape(-1, 784).astype('float32') / 255.0
x_test = x_test.reshape(-1, 784).astype('float32') / 255.0

model.fit(x_train, y_train, epochs=5, batch_size=32, validation_split=0.1)

# 4. 评估
loss, accuracy = model.evaluate(x_test, y_test)
print(f"测试准确率: {accuracy:.4f}")

# 5. 预测
predictions = model.predict(x_test[:5])
print(f"预测结果: {predictions.argmax(axis=1)}")
```

## 附录：PyTorch 2.x 新特性 (torch.compile)

```python
import torch

# 传统方式
model = MyModel()
output = model(input)

# 使用 torch.compile 加速 (PyTorch 2.0+)
model = torch.compile(model)  # JIT 编译优化
output = model(input)         # 速度提升 30%+

# 三种优化模式
model = torch.compile(model, mode='default')    # 平衡模式
model = torch.compile(model, mode='reduce-overhead')  # 减少开销
model = torch.compile(model, mode='max-autotune')     # 最大性能
```

**核心优势**：
- ✅ 自动算子融合
- ✅ 内存优化
- ✅ 减少 GPU 内核调用
