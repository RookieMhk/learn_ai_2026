## 📖 今日核心概念速记

### 注意力机制公式

```
Attention(Q,K,V) = softmax(QK^T / √d_k) × V
```

### Transformer三大组件

```
1. 多头注意力 (Multi-Head Attention)
2. 位置编码 (Positional Encoding)
3. 前馈网络 (FFN) + 残差连接 + 层归一化
```

### 三种架构范式

| 类型 | 代表 | 任务 |
|------|------|------|
| Encoder-Decoder | T5, BART | 翻译、摘要 |
| Encoder-Only | BERT | 分类、NER |
| Decoder-Only | GPT, LLaMA, Qwen | 生成、对话 |

## 🎯 今日重点理解

### Q1: Transformer为什么能替代RNN？

```
RNN问题：
• 顺序计算，无法并行
• 长距离依赖容易丢失
• 梯度消失/爆炸

Transformer优势：
• 完全并行，计算效率高
• 全局注意力，建模任意距离
• 残差+LayerNorm，稳定训练
```

### Q2: 为什么需要多头注意力？

```
单头注意力：只能关注一种语义关系

多头注意力：
• 头1：局部语法关系（相邻词）
• 头2：主谓宾关系
• 头3：全局逻辑关联
• 头4：实体共指关系
• ...
```

### Q3: LLM为什么都是Decoder-Only？

```
• 扩展性好：只需预测下一个Token
• 生成能力强：单向注意力天然适合生成
• 工程成熟：推理优化成熟
• GPT-3验证：大力出奇迹
```
