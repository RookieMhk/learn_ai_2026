### 1. 下列哪个方法可以将NumPy数组转换为PyTorch张量？

```
A. torch.tensor()
B. torch.from_numpy()
C. torch.asarray()
D. A和B都可以
```

<details>
<summary>点击查看答案</summary>

**答案：D**

解析：

### 6. 训练循环中，以下哪个顺序是正确的？

```
A. backward() → zero_grad() → step()
B. zero_grad() → forward() → backward() → step()
C. forward() → backward() → zero_grad() → step()
D. forward() → step() → zero_grad() → backward()
```

<details>
<summary>点击查看答案</summary>

**答案：B**

正确顺序：
1. `optimizer.zero_grad()` - 清零梯度
2. `outputs = model(x)` - 前向传播
3. `loss.backward()` - 反向传播
4. `optimizer.step()` - 更新参数
</details>

## 三、编程题（每题15分，共30分）

### 12. 实现PyTorch版本的线性回归训练

请使用PyTorch实现以下功能：
- 创建模拟数据 y = 2x + 1 + noise
- 使用 `nn.Linear` 定义模型
- 使用 SGD 优化器和 MSE 损失
- 完成100轮训练并打印loss

<details>
<summary>点击查看答案</summary>

**参考答案：**

```python
import torch
import torch.nn as nn
import torch.optim as optim

# 数据
x = torch.randn(100, 1) * 10
y = 2 * x + 1 + torch.randn(100, 1)

# 模型
model = nn.Linear(1, 1)
optimizer = optim.SGD(model.parameters(), lr=0.01)
criterion = nn.MSELoss()

# 训练
for epoch in range(100):
    pred = model(x)
    loss = criterion(pred, y)
    
    optimizer.zero_grad()
    loss.backward()
    optimizer.step()
    
    if (epoch + 1) % 20 == 0:
        print(f"Epoch {epoch+1}, Loss: {loss.item():.4f}")

print(f"学习权重: {model.weight.item():.4f}")
print(f"学习偏置: {model.bias.item():.4f}")
```
</details>

## 四、附加题（加分项）

### 14. 解释 `torch.compile()` 的作用和使用场景

<details>
<summary>点击查看答案</summary>

**参考答案：**

**作用：** PyTorch 2.0引入的JIT编译器，将Python代码编译为优化后的图执行

**优势：**
- 训练加速10-30%
- 推理加速2-10倍
- 减少kernel启动开销

**使用：**
```python
model = torch.compile(model)  # 编译模型
```

**注意事项：**
- 首次调用有编译开销
- 不支持动态控制流
- 需要PyTorch 2.0+
</details>
