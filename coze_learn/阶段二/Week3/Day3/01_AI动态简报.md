## 一、大模型领域

### 1.1 DeepSeek V4：全球首个脱离CUDA的顶级大模型

**核心突破：**
- 2026年4月7日，DeepSeek官方宣布V4版本将100%运行在华为昇腾950PR芯片
- 底层从英伟达CUDA全面迁移至华为CANN异构计算框架
- 这是全球首个彻底摆脱CUDA生态依赖的顶级大模型

**技术亮点：**
- 万亿参数MoE架构，上下文窗口从128K扩展至100万Token
- 推理速度较CUDA平台V3版本提升35倍
- 推理成本降低97%，单百万Token成本仅约1元
- 在知识理解、逻辑推理、多模态生成等指标上全面超越国际主流旗舰模型

**战略意义：**
```
过去：模型适配CUDA → 客户买英伟达 → 生态更强 → 难以替代
现在：模型原生CANN → 客户采购昇腾 → 生态成熟 → 打破垄断
```

**黄仁勋警告：** 若DeepSeek首发即跑在华为平台，对美国将是"灾难性事件"
